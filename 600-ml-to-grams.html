2. 146.1 grams of NaCl is dissolved into a 2.0 L solution. What is the molarity of a new solution that occurs when 200. mL of this solution is diluted to 600. ml


To find the molarity of the original solution, we first need to calculate the number of moles of NaCl that were dissolved:

Number of moles = mass / molar mass

Number of moles = 146.1 g / 58.44 g/mol (molar mass of NaCl)

Number of moles = 2.5 mol

Now, we need to find the molarity of the original solution:

Molarity = number of moles / volume (in liters)

Molarity = 2.5 mol / 2.0 L

Molarity = 1.25 M

To find the molarity of the new solution after dilution, we can use the formula:

M1V1 = M2V2

where M1 is the original molarity, V1 is the original volume, M2 is the new molarity, and V2 is the new volume.

We know M1 = 1.25 M and V1 = 200. mL = 0.2 L. We also know that the solution is diluted to 600 mL = 0.6 L.

Now we can rearrange the formula to solve for M2:

M2 = M1V1 / V2

M2 = 1.25 M x 0.2 L / 0.6 L

M2 = 0.42 M

Therefore, the molarity of the new solution after dilution is 0.42 M.
