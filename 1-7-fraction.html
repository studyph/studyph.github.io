19. the sum of two fractions is 13 1/2.if one of the fractions is 7 1/5,what is the other fraction?​


Answer:

6 3/10

Step-by-step explanation:

Let x be the other fraction.

We know that the sum of two fractions is 13 1/2, which can be written as 27/2 in improper fraction form.

So, we can write an equation:

7 1/5 + x = 27/2

First, we need to convert 7 1/5 to an improper fraction:

7 1/5 = (5*7 + 1)/5 = 36/5

Now we can substitute this into the equation:

36/5 + x = 27/2

To solve for x, we need to get it by itself on one side of the equation. We can start by multiplying both sides by 10, which will eliminate the fractions:

10(36/5 + x) = 10(27/2)

72 + 10x = 135

Next, we can isolate x by subtracting 72 from both sides:

10x = 63

Finally, we can solve for x by dividing both sides by 10:

x = 63/10 = 6 3/10

Therefore, the other fraction is 6 3/10.
