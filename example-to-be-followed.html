27. the following are examples of polynomial


1.Monomial- having only one term. For example
   x,2x,45y

   x+y,2x+4y
3.Trinomial-having three terms indicated by + and/or - symbol. For example
   x+y+z,2xy+2xz-2yz
4.Multinomial-having more than 3 terms indicated by + and/or - symbol. For    example:
   2x+2xy+2xyz-3sd, a+b+c+d+e, q+w+e+r+t+y+u,
Note: Even if there is multiplicarion and division involved, it is considered as only one term. For example, 2(x) which is also equal to 2x; it is just one term.
