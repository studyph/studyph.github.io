QUESTION:

2. A ladder that is 16 feet long is leaning against a tree. The bottom part of the ladder is 8 feet from the base of the tree. How far up the tree if the ladder touching the tree?

SOLUTION:

Given:

Ladder = 16 feet (hypothenus)

Bottom part = 8 feet (base)

Height of the tree = Unknown (Height)

Using the pythagorean theorem:

a²+b² = c²

where a and b are the bases and c is the hypothenus:

Solving for the Height (b)

a²+b² = c²

8²+b² = 16²

64+b² = 256

b² = 256 - 64

b² = 192

b = √192

b = 8√3

Therefore , the height of the tree is 8√3 feet
