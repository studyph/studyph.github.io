2. Square root of 104/100


Answer:

(2√26)/10

Step-by-step explanation:

To find the square root of a number without using a calculator:

Divide the number to a perfect square or find the factors where 1 factor is a perfect squareSimplify

Factors of the numbers where at least one is a perfect square:

104 = 4 × 26   (4 is a perfect square)

100 = 10 × 10

Simplify:

= √(104/100)

= √(4 × 26) / √(10 × 10)

= (2√26)/10

8. the length of a rectangle is 5 more than its width. If the area of the rectangle is 104 square meters,what are the measurements of its lenght and width?


Answer:

Length = 13m

Width = 8m

Step-by-step explanation:

Width = x

Length = x + 5

Area = lw = (x)(x + 5)

(x)(x + 5) = 104

x² + 5x = 104

x² + 5x - 104 = 0

Since the 104 is negative, we'll use

(x + _ )(x - _ )

Find the factors of -104 with the sum of 5

13 × 8 = 104

Now since 5 is positive we'll put the positive sign to the larger number and negative to the smaller number to get the negative in -104

+13 × (-8) = -104

(x + 13)(x - 8) = 0

x + 13 = 0

x = -13

x - 8 = 0

x = 8

Try the values of x to the equation (x)(x + 5) = 104

x = - 13

(-13)(-13 + 5) = 104

(-13)(-8) = 104

104 = 104

x = -13 is a root

x = 8

(8)(8 + 5) = 104

(8)(13) = 104

104 = 104

x = 8 is a root

Normally, we use both roots. But since we're talking about measurements, we'll only use the positive one (x = 8m), since it does not make sense to have a negative measurement (x = -13m)

W = x

W = 8m

L = x + 5

L = 8 + 5

L = 13m

10. four small squares are put together to form a bigger square. If the perimeter of the big square id 104 cm, what is the area of each small square​


Answer:

169 square centimeters

Step-by-step explanation:

Since the perimeter is the total of the lengths of all the sides and a square has 4 sides, then

104 cm / 4 = 26cm

Each side of the big square is 26cm and we divide that by 2 because one side would include one side each of the square. So,

26cm/2=13cm

That will be the measurement of one side of a small square. Now using the formula for area of a square

A= s^2

=(13cm)^2

=169 cm^2
