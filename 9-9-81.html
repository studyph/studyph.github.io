Answer:

A. x² – 81

Step-by-step explanation:

The answer is letter A, x² – 81.

We can simply solve this problem by applying the FOIL Method;

(x + 9) (x – 9): First ; x*x = x²

(x + 9) (x – 9): Outer ; x*(-9) = -9x

(x + 9) (x – 9): Inner ; 9*x = 9x

(x + 9) (x – 9): Last ; 9*(-9) = -81

Combine: x²-9x+9x-81

Combine like terms: x²(-9x+9x)-81

= x² – 81

Shorter way:

If the problem is like this (a+b) (a-b) then the faster way to solve it is by skipping 2 parts in FOIL Method and just do the First and the Last but take note, only use this if the equation of your problem is like this: (a+b) (a-b)
