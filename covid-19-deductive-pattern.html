Step-by-step explanation:

Answer:

Deductive reasoning is a logical process that begins with a general statement, or hypothesis, and examines the possibilities to reach a specific, logical conclusion. It is a method of logical inference based on premises which contain generalizations that are assumed to be true and from which a conclusion can be drawn. Examples of deductive reasoning include syllogisms and argumentative essays. In each case, one must make a set of assumptions in order to arrive at the final conclusion. Therefore, deductive reasoning is used to make logical deductions from a set of observations and can be used to reach an overarching conclusion or to recognize and observe patterns.
