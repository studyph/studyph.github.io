6. Find the measure of the interior angle of a: 1.) 20-gon [formula - n-2 180°]2.)Undecagon [formula - n-2 180°]3.)24-gon [formula - n-2 180°]4.)3,4200 - sum of the interior anglesangles [formula - n=s÷180°+2](With solution)​


1. To find the measure of the interior angle of a 20-gon, we use the formula:

Interior angle = (n - 2) x 180° / n

= (20 - 2) x 180° / 20

= 162°

Therefore, the measure of the interior angle of a 20-gon is 162°.

2. To find the measure of the interior angle of an undecagon (11-gon), we use the same formula:

Interior angle = (n - 2) x 180° / n

= (11 - 2) x 180° / 11

= 147.27° (rounded to two decimal places)

Therefore, the measure of the interior angle of an undecagon is approximately 147.27°.

3. Similarly, to find the measure of the interior angle of a 24-gon, we use the same formula:

Interior angle = (n - 2) x 180° / n

= (24 - 2) x 180° / 24

= 150°

Therefore, the measure of the interior angle of a 24-gon is 150°.

4. To find the value of each interior angle in a polygon with 3,420° as the sum of its interior angles, we use the formula:

Sum of interior angles = (n - 2) x 180°

3,420° = (n - 2) x 180°

n - 2 = 3,420° / 180°

n - 2 = 19

n = 21

Therefore, the polygon has 21 sides. To find the measure of each interior angle, we use the same formula as before:

Interior angle = (n - 2) x 180° / n

= (21 - 2) x 180° / 21

= 160°

Therefore, each interior angle of the polygon measures 160°.

Step-by-step explanation:

In order to find what the set is, we must define the pattern and its restrictions. Set builder notation is written as follows:

{ type of number | condition }

example: { x ∈ R | x > -2 }

The above notation is read as x is an element of real numbers such that x is greater than -2

Notice, that all numbers are multiples of 2 and is within the range from 2 until 180. Hence, the set builder notation can be one of the following:

{2x ∈ N | 1 ≤ x ≤ 90}
