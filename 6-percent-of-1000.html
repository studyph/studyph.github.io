10. please help / the enrolment in a certain private school year 2005 to 2006 was 1250 for school year 2006 2007 the enrolment was only 1000 what was the percent decrease in enrolment? (w solution) grade 6​


Answer:

20%

Step-by-step explanation:

percentage decrease = (decrease / initial value) x 100%

where:

decrease = initial value - final value

Solution:

initial value = 1250

final value = 1000

decrease = 1250 - 1000

decrease = 250

percentage decrease = (250 / 1250) x 100%

percentage decrease = 20%
