Alphanumeric Keypad

An alphanumeric keypad is a keyboard that contains both numbers and letters on the same keys. Typically, they are found on telephones and cellular phones. They also can appear on laptops, ATMs or any device where both numbers and letters are equally necessary.

Numeric Keypad

A separate set of keys on some keyboards that contain the numbers 0 through 9 and a decimal point arranged as on an adding machine. Numeric keypads make it easier to enter large amounts of numeric data. Frequently, the keys on the numeric keyboard also serve as cursor control keys.

Arrow Keys

Arrow keys or cursor movement keys are buttons on a computer keyboard that are either programmed or designated to move the cursor in a specified direction. ... Arrow keys are commonly used for navigating around documents and for playing games.

Control Keys

The Control key on a computer keyboard is a key that is used by pressing it in combination with other keys, enabling other keys on the keyboard to perform secondary functions. It is generally labeled as Ctrl.

Function Keys

The function keys or F keys are lined across the top of the keyboard and labeled F1 through F12. These keys act as shortcuts, performing certain functions, like saving files, printing data, or refreshing a page. For example, the F1 key is often used as the default help key in many programs.

What is the different parts of the computer keyboard?

A keyboard is typewriter like device, which is considered as an input device of the computer. It gathers commands and letters input from the user to execute a process or type a document. A keyboard contains letters from the English alphabets, numbers and symbols. Listed below are the known parts of the keyboard:

Alphanumeric keys - this is the biggest part of the keyboard which could be found at the center of the keyboard. It contains letters from English alphabets with numbers from 0 to 9 at the top. Function keys - functions keys are composed of F1 to F12 keys that corresponds to some system commands and shortcuts when you pressed it on the keyboard. Numeric keys - some keyboards has separate number keys which often found in the right portion of the keyboard. Command keys - these are special keys that are not alphabets, numbers and part of the function keys which would do something when you pressed it on the keyboard.
