ADDING AND SUBTRACTING POLYNOMIALS

[tex]__________________________[/tex]

1.) (-x² - 2) + (2x² + 1)

= -x² - 2 + 2x² + 1

= x² - 2 + 1

= x² - 1

2.) (2x² + 3) - (3 - 4x²)

= (2x² + 3) + (-3 + 4x²)

= 2x² + 3 - 3 + 4x²

= 6x² + 3 - 3

= 6x²

3.) (2x³ + 3x²) - (x³ + 8)

= (2x³ + 3x²) + (-x³ - 8)

= 2x³ + 3x² - x³ - 8

= x³ + 3x² - 8

7. Complete the worksheet below with what have you learned regarding operations on functions. Write your own definition and steps on performing each functions operation. You may give your own example to better illustrate your point. Addition Subtraction Multiplication Division Composition​


OPERATION ON FUNCTION

1. Addition

Assume we have two functions, f(x) and g(x) . The sum of these two functions is defined as,

(f + g)(x) = f(x) + g(x)

Example:

Consider f(x) = 3x + 1 and g(x) = x + 3

(f + g)(x) = f(x) + g(x)f(x) + g(x) = 3x + 1 + x + 3f(x) + g(x) = 4x + 4

2. Subtraction

Assume we have two functions, f(x) and g(x) . The differentiation between these two functions can be expressed as follows:

(f - g)(x) = f(x) - g(x)

Example:

Consider f(x) = 3x + 1 and g(x) = x + 3

(f - g)(x) = f(x) - g(x)f(x) - g(x) = 3x + 1 - (x + 3)f(x) - g(x) = 3x + 1 - x - 3f(x) - g(x) = 2x - 2

3. Multiplication

Assume we have two functions, f(x) and g(x) (x). The product of these two functions is defined as,

(f . g)(x) = f(x) . g(x)

Example:

Consider f(x) = 3x + 1 and g(x) = x + 3

(f . g)(x) = f(x) . g(x)f(x) . g(x) = (3x + 1)(x + 3)  f(x) . g(x) = 3x·x + 3x·3 + 1·x + 1·3f(x) . g(x) = 3x² + 9x + x + 3f(x) . g(x) = 3x² + 10x + 3

4. Division

Assume we have two functions, f(x) and g(x) (x). The division of these two functions is defined as,

(f / g)(x) = f(x) / g(x), where g(x) ≠ 0

where x belongs to both f and g's domains We must specify g(x) ≠ 0 because we cannot divide by zero.

Example:

Consider f(x) = 3x + 1 and g(x) = x + 3

(f / g)(x) = f(x) / g(x)f(x) / g(x) = (3x + 1)/(x + 3)

22. Search for an example of a worksheet. Explain the different columns included in the worksheet.


Accounting Worksheet Columns Included:

-Unadjusted Trial Balance

-Adjusting Entries

-Adjusted Trial Balance

-Income Statement

-Balance Sheet

Explanations:

Unadjusted Trial Balance

A column contains all the company’s assets, liability, expenses, and revenue accounts, which are used in a concerned year.

Adjusting Entries

All the accounts of the company, which requires the passing of the adjustment entry will get listed in the adjustment column.

Adjusted Trial Balance

The adjusted trial balance will be prepared by combining the entries of the previous two columns, Unadjusted trial balance, and adjustments.

Income Statement

A column contains the values concerning the expenses and revenue accounts only. In this case, if the value of total revenue exceeds the expense column, then the difference will be the net income of the year of the company as it is earning more revenues in the year than what it is spending on its expenses.

Balance Sheet

A column contains the values with respect to the assets, liabilities, and owner’s capital only.
