18. A ladder is leaning against a wall. The foot of the ladder is 6.5 feet from the wall. The ladder makes an angle of 74° with the level ground. How high up the wall does the ladder reach?


Answer:

The height of the wall that the ladder reaches can be calculated using trigonometry. The ladder, the ground and the wall form a right triangle. The distance between the foot of the ladder and the wall is the adjacent side of the angle between the ladder and the ground. The height of the wall that the ladder reaches is the opposite side of this angle. We can use the tangent function to find the length of the opposite side: tan(74°) = opposite / 6.5. Solving for opposite, we get opposite = 6.5 * tan(74°) ≈ 23.4 feet. So, the ladder reaches approximately 23.4 feet up the wall.

Answer:

18.7 feet

Step-by-step explanation:

We can use trigonometry to solve this problem. Let's call the height up the wall that the ladder reaches "x". Then, we can use the sine function to relate the angle of 74° to the opposite side (x) and the hypotenuse (the length of the ladder).

sin(74°) = x / length of ladder

We know that the length of the ladder is the distance from the foot of the ladder to the wall, plus the height up the wall that the ladder reaches. So we can substitute in 6.5 + x as the length of the ladder.

sin(74°) = x / (6.5 + x)

To solve for x, we can cross-multiply:

sin(74°) * (6.5 + x) = x

Now we can solve for x by isolating it on one side:

x = sin(74°) * 6.5 / (1 - sin(74°))

Using a calculator, we get:

x = 18.7 feet

So the ladder reaches a height of 18.7 feet up the wall.
