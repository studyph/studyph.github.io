What is project monitoring?

This stage of the project involves keeping a close eye on the entire project management life cycle, ensuring the project activities are on the right track, and comparing the actual performance to the project goals.

Why is Monitoring necessary in a project?

Monitoring activities in a project is essential to ensure that everything is going as planned. Also, to be successful in a project, it is necessary to monitor the proper execution of the various tasks assigned to the member of the project and the results of each milestone. The project monitoring also makes it possible to calculate the budget at any time, based on the time that has already elapsed.

Project monitoring is helpful to mitigate the risk of failure in the project, but sure, Monitoring does not avoid problems. Still, it makes it easier for the people in the project to detect the issues and react quickly to find an appropriate solution at a suitable time.

Who is doing the Monitoring?

People who are in charge of monitoring the progress while the NSTP community immersion project is running are Trainees and Community partners.

This monitoring can be used as material for consideration of what adjustments need to be implemented and adjusted within the project implementation period.

How important is NSTP Project Monitoring Significance?

Monitoring the NSTP project is very important in order to be able to see and determine:

- Whether the activities were carried out according to the schedule;

- Whether the activities were completed on time;

- Whether the outputs were produced on time;

- Whether the outputs were produced in the appropriate amount and quality as specified on the project design.
