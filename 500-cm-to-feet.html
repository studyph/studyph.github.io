Answer :Perimeter : sum of the length of all sides.Area : Square = S²; Rectangle = LW

2. Given : Square , side (S) = 5m

Perimeter = 4S

= 4 (5)

Perimeter = 20 m.

Area = S²

= 5²

Area = 25 m².

3. Given : Rectangle, Area = 500 ft², Width = 20 ft

Find the length (L).

Area = LW

500 = L (20)

divide both sides by 20

Length = 25 ft.

find the Perimeter.

Perimeter = 2L + 2W

= 2 (25) + 2 (20)

= 50 + 40

= 90

Perimeter = 90 ft.

4. Given: Rectangle, Perimeter = 56 m, Length = 16 m

Find the width (W).

Perimeter = 2L + 2W

56 = 2(16) + 2W

56 = 32 + 2W

56 - 32 = 2W

24 = 2W

divide both sides by 2

Width = 12 m.

Find the area.

Area = LW

= 16 × 12

= 192

Area = 192 m².

5. Given: Rectangle, Width(W) = 120 cm, Length = 3W

Find the Length.

Length = 3W

= 3(120)

= 360

Length = 360 cm.

Perimeter.

Perimeter = 2L + 2W

= 2(360) + 2(120)

= 720 + 240

= 960

Perimeter = 960 cm.

Area.

Area = LW

= 360 × 120

= 43 200

Area = 43,200 cm².
