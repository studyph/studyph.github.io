Answer:

To calculate the numbers of subatomic particles in an atom, use its atomic number and mass number: number of protons = atomic number. number of electrons = atomic number.

How many neutrons does Mercury 201 have?

We see that for mercury the atomic number is 80. The atomic number is the number of protons in an atom's nucleus, so we can tell right away that an atom of mercury contains 80 protons. The mass number (201) is the total number of protons and neutrons. So, there must be 201 - 80 = 121 neutrons.\
