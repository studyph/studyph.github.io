8. 1/9 of the shirts sold at Peter's shop are striped. 5/8 of the remainder are printed. The rest of the shirts are plain colored shirts. If Peter's shop has 81 plain colored shirts, how many more printed shirts than plain colored shirts does the shop have?


Total Shirts: x
Striped shirts: 1/9 (x)
Remainder:  8/9 (x)
Printed: 5/8 (8/9)(x)  or 5/9(x)
Plain colored shirts: 81

Find the total, x:
x - [1/9(x) + 5/9 (x)] = 81
x - 6/9(x) = 81
9x - 6x =(81)(9)
3x/3 = 729/3
x = 243

Striped Shirts = 1/9(243)
                      = 27

Printed Shirts = 5/8 (8/9) (243)
                       = 135

Printed shirts - plain colored shirts = 135 - 81
                                                       =  54

The shop has 54 more printed shirts that plain colored shirts.
