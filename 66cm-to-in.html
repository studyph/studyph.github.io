11. A rectangle has length of 5 more than it's width The area of the rectangle is 66cm²,find the length and the width of the rectangle.​


Answer:

Length = 11cm

Width = 6cm

Step-by-step explanation:

Let x be the width

Let x+5 be the length

Area = LW

66cm² = x(x+5)

x²+5x = 66

x² + 5x - 66 = 0

(x + 11)(x - 6) = 0

x = -11 & 6

x must be greater than 0, therefore x = 6

Length = 6 + 5 = 11cm

Width = 6cm
