4. how infinity mirror works? own words.​


Answer:

An infinity mirror creates an optical illusion of an infinitely long tunnel. It actually consists of two mirrors — a regular mirror and a one-way mirror — with light emitting diodes (LEDs) sandwiched between them. When the LEDs are turned off, the infinity mirror acts just like a regular mirror.

Also, when light is reflected from a mirror, it bounces off at the same angle in the opposite direction from which it hit. For example, if the light hits a flat or "plane mirror" at a 30-degree angle from the left, it will bounce off at a 30-degree angle to the right.

15. How much is infinity x 0=???


Answer:

For the umpteen-trillionth time, infinity isn’t a number.

So it can’t be a value that’s subject to arithmetic operations, such as multiplication.

However, it can function as a limit to an arithmetic value, such as in this case:

For any real number r, not equal to zero, the value of r/x, as x approaches infinity, is zero.

Let’s see how we can rephrase your question:

For any real number r, what is the value of 0 * r, as r approaches infinity?

The answer is zero.

What is the result of dividing a number by zero and adding infinity to it ?

୨ ANSWER ୧

The result of dividing any non-zero number by zero is undefined ( or sometimes expressed as " infinity " in certain contexts ). Adding infinity to an undefined result still gives an undefined result. Therefore, the result of dividing a number by zero and adding infinity to it is undefined. In mathematics, division by zero is generally considered an invalid operation because it leads to contradictions and inconsistencies.
