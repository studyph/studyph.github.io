25. During an initial phase of an outbreak of coronavirus, the number of infections can grow geometrically. if there were 2 infections in the first day, 4 in the second day, and 8 in the third, how many will be infected on the 10th day?


The amount of people that will be infected on the 10th day is 1024.

We must not confuse exponential growth and geometric growth. To calculate the geometric growth or geometric progression, we must follow these steps.

Step 1: List all the given variables

A1=2

A2=4

A3=8

Step 2: Identify the common ratio

4/2 = 2

8/4 = 2

Therefore the commo ratio is 2.

There are two ways we can go about solving this problem.

1st Way:

Since the numbers involved are not that big we can calculate it by one by one since we now know that we only need to multiply the number by 2 to get the next value.

A3=8

A4=8 x 2 = 16

A5=16 x 2 = 32

A6=32 x 2 = 64

A7=64 x 2 = 128

A8=128 x 2 = 256

A9=256 x 2 = 512

A10=512 x 2 = 1024

2nd Way

Use the geometric growth formula:

an=a1r^n-1

Where a1 = the first value

an = value of the nth term

r= common ratio

Plug in the numbers:

a10=(2)(2)^10-1

a10 = 1024
