25. prove [tex]\frac{sec (x)-csc(x)sec(x)+csc(x)}{tanx+secx-1} = \frac{sin(x)cos(x)+cot^2(x)sin(x)cos(x)}{1+sin(x)}[/tex]


Answer:

[tex]\sf Let \ L.H.S = \frac{sec(x) - csc(x)sec(x)+csc(x)}{tan(x)+sec(x)-1}, R.H.S = \frac{sin(x)cos(x)+cot^2(x)sin(x)cos(x)}{1+sin(x)}[/tex]

ㅤ

[tex]\sf L.H.S = \frac{sec(x)-csc(x)sec(x)+csc(x)}{tan(x)+sec(x)-1}[/tex]

[tex]\implies \sf L.H.S = \frac{\frac{sin(x)}{cos(x)} \times \frac{1}{sin(x)} - csc(x)sec(x) + csc(x) }{tan(x)+sec(x)-1}[/tex]

[tex]\implies \sf L.H.S = \frac{tan(x)csc(x) - csc(x)sec(x)+csc(x)}{tan(x)+sec(x)-1}[/tex]

[tex]\implies \sf L.H.S = \frac{(csc(x))(tan(x)-sec(x) + 1)}{tan(x)+sec(x)-1}[/tex]

[tex]\implies \sf L.H.S = \frac{(csc(x))(tan(x)-sec(x)+1)}{tan(x)+sec(x)-(sec^2(x)-tan^2(x))}[/tex]

[tex]\implies \sf L.H.S = \frac{(csc(x))(tan(x)-sec(x)+1)}{[tan(x)+sec(x)][1-(sec(x)-tan(x))]}[/tex]

[tex]\implies \sf L.H.S = \frac{(csc(x))(tan(x)-sec(x)+1)}{[tan(x)+sec(x)](tan(x)-sec(x)+1)}[/tex]

[tex]\implies \sf L.H.S = \frac{csc(x)}{tan(x)+sec(x)}[/tex]

[tex]\implies \sf L.H.S = \frac{csc(x)}{\frac{sin(x)}{cos(x)}+\frac{1}{cos(x)} }[/tex]

[tex]\implies \sf L.H.S = \frac{csc(x)}{\frac{sin(x)+1}{cos(x)} }[/tex]

[tex]\implies \sf L.H.S = \frac{csc(x)cos(x)}{sin(x)+1}[/tex]

[tex]\implies \sf L.H.S = \frac{\frac{1}{sin(x)}cos(x) }{sin(x)+1}[/tex]

[tex]\implies \sf L.H.S = \frac{\frac{cos(x)}{sin(x)} }{sin(x)+1}[/tex]

[tex]\implies \sf L.H.S = \frac{cot(x)}{sin(x)+1}[/tex]

ㅤ

[tex]\sf R.H.S = \frac{sin(x)cos(x)+cot^2(x)sin(x)cos(x)}{1+sin(x)}[/tex]

[tex]\implies \sf R.H.S = \frac{(cos(x))(sin(x)+cot^2(x)sin(x))}{1+sin(x)}[/tex]

[tex]\implies \sf R.H.S = \frac{(cos(x))(sin(x)(1 + cot^2(x)))}{1+sin(x)}[/tex]

[tex]\implies \sf R.H.S = \frac{cos(x)sin(x) \times csc^2(x)}{1+sin(x)}[/tex]

[tex]\implies \sf R.H.S = \frac{cos(x)sin(x) \times \frac{1}{sin^2(x)} }{1+sin(x)}[/tex]

[tex]\implies \sf R.H.S = \frac{cos(x) \times \frac{1}{sin(x)} }{1+sin(x)}[/tex]

[tex]\implies \sf R.H.S = \frac{\frac{cos(x)}{sin(x)} }{1+sin(x)}[/tex]

[tex]\implies \sf R.H.S = \frac{cot(x)}{1+sin(x)}[/tex]

ㅤ

[tex]\therefore \sf L.H.S = R.H.S[/tex]

27. prove the identity tan²x + 2 tanx cotx + cot²x = sec²x csc²x p.s. need answer immediately ​


Proof:

Given: tan²x + 2 tanx cotx + cot²x = sec²x csc²x

We will use the trigonometric identities to prove the given identity.

tan²x + 2 tanx cotx + cot²x

= tan²x + (2tanx/tanx)cotx + cot²x

= tan²x + (2/1)cotx + cot²x

= tan²x + cotx + cot²x

= (tanx + cotx)(tanx + cotx)

= (secx)(cscx)(secx)(cscx)

= sec²x csc²x

Hence, the given identity is proved.
