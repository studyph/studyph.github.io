One of the measures of the central tendency that is easily solved is the Median. The median is simply the mid-value of the given data set or array. Measures of Central Tendency are somehow important as they represent the characteristics of a certain population.

Measures of Central Tendency

1. Mean

2. Median

3. Median

Steps in Solve for the Median

1. Arrange the data set in either ascending or descending order.

2. Simply take the middle value for an odd number of objects.

3. Take the average of the two middle values for an even number of objects.

12. What is the median of the following numbers? (10, 39, 71, 39, 76, 38, 25)


The median of the given data set is 39.

Step-by-step explanation:

The median is one of the measures of central tendency in statistics. The median is the central number of a data set after it is sorted from smallest to largest, and vice versa. If there are 2 numbers in the middle, the median is the average of those 2 numbers.

How to find the median:

Arrange data values from lowest to highest valueSimply take the middle value for an odd number of objects.Take the average of the two middle data values for an even number of objects.

We can use the following formula to calculate the median:

If N is odd:

[tex]Median=(\frac{N+1}{2})[/tex]

If N is even:

[tex]Median = \frac{(\frac{N}{2})+(\frac{N}{2}+1 )}{2}[/tex]

Given: 10, 39, 71, 39, 76, 38, 25

To find: The median of the following number.

Solution:

Arrange data values from lowest to highest value

10, 25, 38, 39, 39, 72, 76

There are 7 samples, therefore N = 7

Calculate Median using the formula given above

[tex]Median=(\frac{N+1}{2})[/tex]

             [tex]= (\frac{7+1}{2})[/tex]

             [tex]=\frac{8}{2}[/tex]

             [tex]=4[/tex]

Median = 4th sample.

10, 25, 38, 39, 39, 72, 76

Hence, the median of the given data set is 39.

What is the median of the following numbers 10, 39, 71, 42, 39, 76, 38, and 25?

- Median is defined as the middle value of a sorted list of numbers. The middle number is found by arranging the numbers. The numbers are arranged in ascending order. Once the numbers are ordered, the middle number is called the median of the given data set.

- Median for Odd data

To find the median for even data, we can use this formula :

Let x1, x2, x3, ..., xn is meaning the data in ascending order.

Median = [tex]x_{\frac{n+1}{2} }[/tex]

- Median for Even data

To find median for even data, we can using this formula :

Let x1, x2, x3, ..., xn is meaning the data in ascending order.

Median = [tex]\frac{(x_{\frac{n}{2} } + x_{\frac{n+2}{2} } )}{2}[/tex]

10, 25, 38, 39, 39, 42 , 71, 76

Total data is 8, because 8 is even so me using the median for even data, and we get :
