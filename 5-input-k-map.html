2. A function is a relation where the input is mapped to ____ output.


A function is a relation where the input is mapped to range output.

Explanation:

Relations and functions define a mapping between two sets (Inputs and Outputs) such that they have ordered pairs of the form (Input, Output). The first coordinate of an ordered pair in a relation is the input, and the second coordinate is the output. A function maps each input to one and only one output. Relation and function are very important concepts in algebra. They are used widely in mathematics as well as in real life.

A relation has an input value which corresponds to an output value. When each input value has one and only one output value, that relation is a function. Functions can be written as ordered pairs, tables, or graphs. The inputs and outputs of a function are also called the domain and range. The domain of a function is the set of all input values. The range of a function is the set of all output values.
