2. explain the core competencies and strategic work system design​


Core competencies refer to the unique strengths or capabilities that a company possesses that give it a competitive advantage in the market. These can include things like proprietary technology, specialized expertise, a strong brand, or efficient internal processes.Strategic work system design refers to the process of creating an organizational structure and set of processes that align with a company's core competencies and overall business strategy. This can include things like creating cross-functional teams, streamlining communication and decision-making processes, and implementing performance metrics that align with the company's goals. The goal of strategic work system design is to create a work environment that is efficient, effective, and conducive to achieving the company's strategic objectives.

22. why are operating systems designed to work only with a specific device/technology?​


Answer:

Operating systems are designed to work with specific devices or technologies because different devices and technologies have unique hardware components and software interfaces that require specific instructions to interact with the operating system. In order for an operating system to function correctly and efficiently, it must be designed to communicate effectively with the specific hardware components and software interfaces of a particular device or technology. Creating an operating system that is capable of working with all devices and technologies would be incredibly complex and impractical, as each device and technology has its own unique features and requirements. Therefore, operating systems are designed to work only with specific devices or technologies to ensure compatibility and optimal performance.
