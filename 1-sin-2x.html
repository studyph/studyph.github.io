1. prove (1-cot^2x/1+cot^2x) = sin^2x - cos^2x​


We can start with the left-hand side of the equation and use the identity for cotangent:

(1 - cot^2x) / (1 + cot^2x)

= [(1 - cos^2x/sin^2x) / (1 + cos^2x/sin^2x)] (since cot^2x = cos^2x/sin^2x)

= [(sin^2x - cos^2x) / sin^2x * (1 + cos^2x/sin^2x)] (multiplying numerator and denominator by sin^2x)

= [(sin^2x - cos^2x) / (sin^2x + cos^2x)] * [(sin^2x + cos^2x) / sin^2x]

= (sin^2x - cos^2x) / sin^2x

= sin^2x / sin^2x - cos^2x / sin^2x

= 1 - cos^2x / sin^2x

= sin^2x - cos^2x

Therefore, we have shown that (1 - cot^2x) / (1 + cot^2x) = sin^2x - cos^2x.
