22. $200 x+ $150Y ≥ $1000 ​


Answer:

To solve this inequality, we need to isolate the variable $x$ on one side and the constant value $1000$ on the other side. To do this, we can first subtract $150y$ from both sides of the inequality to get $200x + 150y - 150y \geq 1000 - 150y$. This simplifies to $200x \geq 1000 - 150y$.

Next, we can divide both sides of the inequality by 200 to get $x \geq \frac{1000 - 150y}{200}$. Since division by a positive number does not change the direction of the inequality, the final solution to this inequality is $x \geq \frac{1000 - 150y}{200}$.

To check this solution, we can plug in some sample values of $x$ and $y$ and make sure that the inequality is satisfied. For example, if we plug in $x = 2$ and $y = 1$, we get $2 \geq \frac{1000 - 150 \cdot 1}{200}$, which simplifies to $2 \geq \frac{850}{200}$, or $2 \geq 4.25$. This is true, so the solution is correct.

Note that this inequality has an infinite number of solutions because the variable $x$ can take on any value that is greater than or equal to $\frac{1000 - 150y}{200}$.

PERCENTAGE, BASE, AND RATE

In your given problem, we're asked to find the rate.

The percentage is the answer when you multiply the rate and base, base is the whole number in the problem, and the rate is the number with a percent (%) sign.

Additional:

To convert a decimal into a percentage, multiply the decimal by 100.

To find the rate, we will use the formula:

[tex]\boxed{\rm{R = P \div B}}[/tex]

[tex]\\[/tex]

[tex]\rm{R = P \div B}[/tex][tex]\rm{R = 200 \div 1,000}[/tex][tex]\rm{R = 0.2}[/tex][tex]\rm{R = 0.2 \times 100}[/tex][tex]\rm{R = \green{20 \%}}[/tex]
