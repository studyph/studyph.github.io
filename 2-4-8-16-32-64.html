Example:
Find the 1st, 2nd, 3rd and 4th term using the general rule an = 2(2n).

For the 1st term. Substitute n with 1.

an = 2(2n)
a(1) = 2 (2 (1))
a(1) = 2 (2)
a(1) = 4

2nd term. Substitute n with 2.

an = 2(2n)
a(2) = 2 (2 (2))
a(2) = 2 (4)
a(2) = 8

3rd term. Substitute n with 3.

an = 2(2n)
a(3) = 2 (2 (3))
a(3) = 2 (6)
a(3) = 12

4th term. N = 4.

an = 2(2n)
a(4) = 2 (2 (4))
a(4) = 2 (8)
