20. A SITE A ABC, LS = 40°, LI = 55° AB=3x+2 cm and 3I = 11cm. a. Find The value of x b. Find the m<c ​


Answer:

We can use the Law of Sines to solve for x and m<c.

a. Using the Law of Sines, we have:

sin(40°)/AB = sin(55°)/BC

where BC = 3I/3 = 11/3 cm

Substituting AB = 3x+2 cm and simplifying, we get:

sin(40°)/(3x+2) = sin(55°)/(11/3)

Cross-multiplying and simplifying, we get:

sin(40°) * (11/3) = sin(55°) * (3x+2)

Dividing both sides by sin(40°), we get:

(11/3) * (sin(55°)/sin(40°)) = 3x+2

Simplifying, we get:

x = [(11/3) * (sin(55°)/sin(40°)) - 2]/3

Using a calculator, we get:

x ≈ 1.28 cm

b. To find m<c, we can use the Law of Cosines:

c² = a² + b² - 2ab*cos(c)

where a = AB = 3x+2 cm, b = BC = 11/3 cm, and c = m<c

Substituting and simplifying, we get:

c² = (3x+2)² + (11/3)² - 2(3x+2)(11/3)*cos(m<c)

Dividing both sides by (11/3)² and simplifying, we get:

(9/121) * c² = (9x² + 12x + 4)/121 + 1 - (22x+44/33)*cos(m<c)

Multiplying both sides by (121/9) and simplifying, we get:

c² = (9x² + 12x + 4) + (121/9) - (22x+44/33)*(121/9)*cos(m<c)

Substituting x ≈ 1.28 cm and simplifying, we get:

c² ≈ 16.12 - 38.57*cos(m<c)

Using a calculator, we can solve for cos(m<c):

cos(m<c) ≈ (16.12 - c²)/38.57

Substituting c = 11/3 cm and simplifying, we get:

cos(m<c) ≈ 0.569

Taking the inverse cosine, we get:

m<c ≈ 55.6°

Therefore, the value of x is approximately 1.28 cm and the measure of angle <C is approximately 55.6 degrees.
