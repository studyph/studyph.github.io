Answer:

The number is 6049.

Step-by-step explanation:

How did we get 6049?

To find the mystery number, it is important that you know first the different place value of the numbers.

Place Value

The place value of a digit is represented by the position of a digit in the number.

Every number has its place value. These are the names of each place value arranged from the highest to lowest place value.

Millions Hundred-Thousands Ten-ThousandsThousandsHundredsTensOnes

Examples:

Place value of 8 in every number.

138 - ones

286 - tens

12,890 - hundreds

8,100 - thousands

81,234 - ten thousands

809,600 - hundred thousands

8,092,130 - millions

Now, going back to the mystery number above. We need to identify and arrange the digits according to the given clues and its place value.

Let us do it one-by-one.

"My hundreds digits means nothing."

Nothing means zero.

hundreds = 0

"Six more than my hundreds digit is my thousands."

Add 6 to hundreds which is 0.

thousands = 6 + 0 = 6

"My tens digit is four more than my hundreds."

Add 4 to hundreds which is 0.

tens = 4 + 0 = 4

"My ones digit is five more than tens digit."

Add 5 to tens which is 4.

ones = 5 + 4 = 9

Now that we know every digits. Let us arrange the digits in order according to their place value.

thousands hundreds tens ones

6 0 4 9

The mystery number is 6409.

Value

The value of every digits in a number depends on its place value.

Examples:

Value of 8 in every number.

138 (ones) - 8

286 (tens) - 80

12,890 (hundreds) - 800

8,100 (thousands) - 8,000

81,234 (ten thousands) - 80,000

809,600 (hundred thousands) - 800,000

8,092,130 (millions) - 8,000,000

Expanded Notation

We can express numbers using the place value and value of every digits. It is shown as a sum of each digit multiplied by its matching place value.

Examples:

4,265

= (4 × 1,000) + (2 × 100) + (6 × 10) + (5 × 1)

345,789

= (3 × 100,000) + (4 × 10,000) + (5 × 1,000) + (7 × 100) + (8 × 10) + (9 × 1)

24. meaning of round number nearest ten thousand (pls magbigay ng example)​


The rule for rounding to the nearest ten thousand is to look at the last four digits. If the last four digits are 5,000 or greater, then we round our ten thousands digit up, and if it is less than 5,000, then we keep our ten thousands digit the same.

For example, 43,567 round to the nearest ten thousand. The last four digits are less than 5,000 so 43,567 is rounds down to 40,000.
