Step-by-step explanation:

If Mr. Holloway makes the guitars at the same rate as his last order, it will take him the same amount of time to make 7 guitars as it did to make 2 guitars.

We know that it took 8 weeks to make 2 guitars, so to find out how long it will take to make 7 guitars, we can use the formula:

Time to make 7 guitars = (Time to make 2 guitars) x (Number of guitars / 2)

So the time to make 7 guitars is:

Time to make 7 guitars = 8 weeks x (7 guitars / 2 guitars)

Therefore, it will take Mr. Holloway 8 x (7/2) = 8 x 3.5 = 28 weeks to make the 7 guitars.
