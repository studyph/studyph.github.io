3. What do you think, the positive and negative effects of a keyboard that is arrange alphabetically?​


As the QWERTY keyboard is considered the world standard at this stage, you benefit from having a wide variety of designs to choose from if you use this keyboard. Some come with special, ergonomic shapes, others are designed specifically for office, web or gaming use. All computer retailers stock QWERTY keyboards, while you might have to really search to find a Dvorak or Colemak keyboard in stores. Because of the popularity of the QWERTY design, it has started to replace the numeric keypad on mobile phones. Almost any device you encounter comes standard with a QWERTY keyboard, making it a very familiar layout.

Learning Curve and Muscle Memory
If you are touch typing on a QWERTY keyboard, you are making use of muscle memory, which increases your speed. Switching over to a Dvorak keyboard, for instance, will feel awkward and you will have to search for the keys at first. However, some users claim that the Dvorak has a shorter learning curve than the QWERTY if are learning to type from scratch. The Colemak was designed to be more efficient than the QWERTY, but with a short learning curve if you are switching over from QWERTY to Colemak, as the locations of only 17 keys were changed. If you decide to switch over to another layout, you might struggle to type QWERTY again, and will end up frustrated when trying to work at other people's computers.

26. Use the cyclical alphabet encrypting code that shifts each letter 17 position to a.encode ALPINE SKIING b.decode TIFJJ TFLEKIPJBZZEX


It's ALPINE SKIING and CROSS COUNTRY SKIING

The given words must be encoded using the cyclic alphabetic encrypting code with a shift of 17 letters. This is also known as Caesar Cipher coding, in which we shift our alphabet with the required shift positions.

Let us encode the questions we've been given.

(a) ALPINE SKIING

Now using a Shift of 17 positions, we encode it as,

A = +17 = RL = +17 = CP = +17 = GI = +17 = ZN = +17 = EE = +17 = VS = +17 = JK = +17 = BI = +17 = ZI = +17 = ZN = +17 = EG = +17 =  X

Therefore,

ALPINE SKIING = RCGZEV JBZZEX   ( Answer for Part a )

(b) CROSS COUNTRY SKIING

Now, again we are doing a position shift of +17 for the given words as done in part (a) above, and we get the output below,

T = +17 = CI = +17 = RF = +17 = OJ = +17 = SJ  = +17 = ST = +17 = CF = +17 = OL = +17 = UE = +17 = NK = +17 = TI = +17 = RP  = +17 = YJ = +17 = SB = +17 = KZ = +17 = IZ = +17 = IE = +17 = NX = +17 = G

Therefore,

TIFJJ TFLEKIP JBZZEX = CROSS COUNTRY SKIING ( Answer for part b)

The Caesar Cipher technique is one of the oldest and most basic methods of encryption. It's simply a substitution cipher in which each letter of a given text is replaced by a letter from a fixed number of positions down the alphabet. With a shift of 1, for example, A would be replaced by B, B would become C, and so on. The method is said to have been named after Julius Caesar, who used it to communicate with his officials.
