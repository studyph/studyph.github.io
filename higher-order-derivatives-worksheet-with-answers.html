16. How do you find the minimum and maximum points of a function, using the first-order derivative test?


Step-by-step explanation:

The first-order derivative test is used to find the minimum and maximum points of a function. To use this test, you must first take the derivative of the function and set it equal to zero. This will give you the critical points of the function. Then, you must evaluate the second derivative at each critical point to determine whether it is a minimum or maximum point. If the second derivative is positive, then it is a minimum point; if it is negative, then it is a maximum point.

21. Find the first order partial derivative with respect to y f(x, y) = x³y² + 3xe​


Answer:

f(x, y) = x³y² + 3xe

Take the partial derivative both sides.

f'(x, y) = ∂ / ∂x(x³y² + 3xe)

Evaluate the derivative.

f'(x, y) = 3x²(0) + 3e

Any number multiplied by zero is always zero.

f'(x, y) = 0 + 3e

Any number added by zero will remain the same.

f'(x, y) = 3e
