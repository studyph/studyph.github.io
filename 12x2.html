10. 5) Which of the following quadratic equation has 2/3and -1/4 as their roots? A. 12x2 + 5x + 2 = 0 B. 12x2 – 5x + 2 = 0C. 12x2 -5x - 2 = 0 D. 12x2 + 5x - 2 = 0​

Step-by-step explanation:

Given (roots):

x = ²/₃

x = -¹/₄

The two factors are (3x - 2) and (4x + 1).

Use the FOIL method:

(3x - 2)(4x + 1)

= (3x)(4x) + (3x)(1) + (-2)(4x) + (-2)(1)

= 12x² + 3x - 8x - 2

= 12x² - 5x - 2

15. 10. Which of the following is the product of (-3x) and (-4x +2y)?A. 12x2 - 6xyB.-12X2 + 6xyC. 12x -ontcinc hcidose​


A. 12x² - 6xy

Step-by-step explanation:

Given: (-3x) (-4x + 2y)

You will use the distributive property of multiplication. And it states that you will need to distribute the -3x to the first number (-4x) and the second number (2y), and multiply by each of them separately.

(-3x) (-4x + 2y)

= (-3x • -4x) (-3x • 2y)

= 12x² - 6xy
