To determine the center and radius of each circle, we need to write each equation in standard form:

1. (X + 5)² + y² = 2²

Standard form: (x - (-5))² + (y - 0)² = 2²

Center: (-5, 0)

Radius: 2

Graph: Blue

2. X² + Y² -4 = 0

Standard form: (x - 0)² + (y - 0)² = 2²

Center: (-3, -2)

Center: (3, -2)

Graph: Green
