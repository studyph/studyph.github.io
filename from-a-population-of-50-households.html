16. 7. From a population of 50 households, in how many ways can a researcher select a sample with a size of 10?​


Answer:

The number of ways to select a sample of size 10 from a population of 50 households can be calculated using the combination formula:

n C r = n! / (r!(n-r)!)

Where n is the population size and r is the sample size.

In this case, we have:

n = 50 (population size)

r = 10 (sample size)

So the number of ways to select a sample of size 10 from a population of 50 households can be calculated as:

50 C 10 = 50! / (10!(50-10)!)

= (50 x 49 x 48 x 47 x 46 x 45 x 44 x 43 x 42 x 41) / (10 x 9 x 8 x 7 x 6 x 5 x 4 x 3 x 2 x 1)

= 10,272,278,170
