[tex]P_{\text{H}_{2}}[/tex]

Solution:

[tex]P_{\text{T}} = P_{\text{H}_{2}} + P_{\text{H}_{2}\text{O}}[/tex]

[tex]P_{\text{H}_{2}} = P_{\text{T}} - P_{\text{H}_{2}\text{O}}[/tex]

[tex]P_{\text{H}_{2}} = \text{742.5 mm Hg} - \text{23.8 mm Hg}[/tex]

[tex]\boxed{P_{\text{H}_{2}} = \text{718.7 mm Hg}}[/tex]

ANSWER: D. 4.63L

We'll use the Combined Gas Law, which is:

We also need to convert the terms to standard units.
15C + 273 = 288K
745 torr ÷ 760 = 0.98 atm
40C + 273 = 313K
700 torr ÷ 760 = 0.92 atm

Pluggin in the terms:
[tex] \frac{0.98(4.00)}{288} = \frac{0.92(x)}{313} \\ \frac{0.98(4.00)(313)}{288(0.92)} = 4.63[/tex]
The gas will occupy 4.63 L.

We have the following information:  

V1 (initial volume) = 4.00 L

V2 (final volume) = ? (in L)  

T1 (initial temperature) = 15ºC (in Kelvin)

1 atm --- 760 torr

x = 745/760

1 atm --- 760 torr

x atm ---- 700 torr

760x = 700

x = 700/760

Now, we apply the data of the variables above to the General Equation of Gases, let's see:

[tex]\dfrac{P_1*V_1}{T_1} =\dfrac{P_2*V_2}{T_2}[/tex]

29. Activity 1Directions: Solve the following problems using Boyle's Law.1. A 5.6 mL of a gas in a piston exert a pressure of 1.7 atm. It is compresseduntil the volume reached 4.8 ml. Calculate the new pressure inside the piston.2. If at 4 atm the volume of a dry gas is 250 ml. What is the volume of the gas ifthe pressure in decreased by 12?3. The gas has a volume of 2.7 ml. What is the initial pressure if volumebecomes 5.8L at 2.atm?​


Boyle's LawP1V1 = P2V2

where in, P1 = first pressure

V1 = first volume

P2 = first pressure

V2 = second volume

1) Given: P1 = 1.7 atm

V1 = 5.6 mL

V2 = 4.8 mL

Unknown: P2 = ?

Formula: P2 = P1V1 / V2

Note: I used the original formula of Boyles Law to get the needed formula for this problem by dividing terms and cancellation of like terms.

P1V1 = P2V2

P1V1 / V2 = P2V2 / V2

P2 = P1V1 / V2

• easier method is by transposition:

e.g. P2 = ?

P1V1 = P2V2 <---- transpose V2 to the other side

P1V1 / V2 = P2 <---- formula for P2

Solution: P2 = P1V1 / V2

= (1.7atm)(5.6mL) / 4.8mL

= 1.98atm

Note: mL is cancelled out, so the unit for pressure will remain

Reminder: cancellation of unit(s) is applicable when the other same unit was in denominator.

Final Answer: P2 = 1.98 atm

2) Given: P1 = 4 atm

V1 = 250 mL

P2 = 12 atm

Unknown: V2 = ?

Formula: V2 = P1V1 / P2

Solution: V2 = P1V1 / P2

= (4atm)(250mL) / 12atm

= 83.33mL

Final Answer: V2 = 83.33 mL

3) Given: V1 = 0.0027 L

P2 = 2 atm

V2 = 5.8 L

Note: conversion is applied, because we're going to cancel units, so we need "the same" unit.

mL -----> L

2.7mL × 1L / 1000mL = 0.0027L

Unknown: P1 = ?

Formula: P1 = P2V2 / V1

Solution: P1 = P2V2 / V1

= (2atm)(5.8L) / 0.0027L

= 4296.30atm

Final Answer: P1 = 4296.30 atm

[tex]P_{\text{H}_{2}\text{O}} = \text{23.8 torr}[/tex]
