7. a red dress priced PHP 13,599.15 for a television set that originally costs 15,999.00 what is the rate of discount?


Answer:

15%

Step-by-step explanation:

Given:

Selling price = 13,599.15Original cost = 15,999.00

Formula to be use

Rate = (OC - SP) / OC × 100%

where OC = Original cost

SP = Selling Price

Plugging the values , we have

Rate = (15,999 - 13,599.15) / 15,999 × 100%

Rate = 2,399.85 / 15,999 × 100%

Rate = 0.15 × 100%

Rate = 15%

Therefore , the rate is 15%
