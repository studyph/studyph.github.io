8. 1.Decimal to binary number 30. 2.Decimal to binary 15 pa help Po Dito module kasi kame Wala teacher kaya di ko gets ​


Answer:

Okay, I understand. Converting between number systems like decimal and binary can be tricky if not explained well. Let me walk you through these examples to help you understand:

1. The decimal number 30 converted to binary is:

30 / 2 = 15 r 0

so the last digit is 0

15 / 2 = 7 r 1

so the digit before last is 1

7 / 2 = 3 r 1

so the third digit is 1

So 30 in decimal is 11112 in binary.

2. The decimal number 15 converted to binary is:

15 / 2 = 7 r 1

so the last digit is 1

7 / 2 = 3 r 1

so the digit before last is 1

3 / 2 = 1 r 1

so the third digit is 1

So 15 in decimal is 1111 in binary.

Some tips for converting between these number systems:

•Divide the number by 2 and take the remainder as the last binary digit. Keep dividing by 2 to get subsequent digits.

•Start from the least significant digit (ones place) and work your way up to the most significant digit.

•Only two digits are used in binary: 0 and 1.

•Converting back to decimal, calculate (2^0)*1 + (2^1)*1 + (2^2)*0 + (2^3)*1 = 30.
