1. A hot air balloon is 1200 feet above the ground. The angle of depression from the balloon to the person holding the string of the balloon is 88 degrees. What is the length of the string from the balloon to the hand of the person holding it?

2. Building A and B are across the street from each other, 35 meter apart. From a point on the roof of Building A, the angle of elevation to the top pf the Building B is 23 degrees, and the angle of depression to the base of Buidling B is 37 degrees. How tall is each building?

To find the length of the string, we can use the tangent function. The tangent of the angle of depression is equal to the opposite side (the height of the person) divided by the adjacent side (the length of the string). Using this relationship, we can write the following equation:

tan(88) = 1200 / x

where x is the length of the string. Solving for x, we get:

x = 1200 / tan(88)

Note: the tangent of 88 degrees is not defined, as the tangent is undefined for angles greater than 90 degrees.

To find the height of each building, we can use similar triangles. Let h1 be the height of Building A and h2 be the height of Building B. Then, we have:

h1 / 35 = h2 / (h2 - h1)

tan(23) = h2 / 35

h2 = 35 tan(23)

And

h1 / 35 = (h2 - h1) / 35

tan(37) = (h2 - h1) / 35

h2 - h1 = 35 tan(37)

Solving for h1 and h2, we can substitute the value of h2 found earlier into the second equation to find h1.
