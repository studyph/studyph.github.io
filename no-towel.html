Let x be the number of suites receiving new towels and y be the number of individual rooms receiving new towels.


Answer:

10x + 6y ≥ 778

Step-by-step explanation:

Define x and y:

Let x be the number of suites receiving new towels.

Let y be the number of individual rooms receiving new towels.

Find the number of towels needed for suites:

one suite = 10 towels

x suites = 10x towels

Find the number of towels needed for individual rooms:

1 individual room = 6 towels

y individual rooms = 6y towels

Given that the minimum number of towels required is 778.

⇒ 10x + 6y ≥ 778

27. three vendors sold towels in the ratio of 2:3:4. Together, they sold 126 towels, How many towels did each vendor sell?​


The ratio of the number of towels sold by the three vendors is 2:3:4. This means that for every 2 towels sold by the first vendor, 3 towels are sold by the second vendor, and 4 towels are sold by the third vendor.

Given that the total number of towels sold is 126, we can use this ratio to find the number of towels sold by each vendor.

First, we need to find the common denominator of 2, 3, and 4, which is 12. So we can express the ratio as 2:3:4 = 2/12:3/12:4/12 = 1/6:1/4:1/3

To find the number of towels sold by each vendor, we multiply the total number of towels sold (126) by the ratio of each vendor.

First Vendor: 126 x 1/6 = 21

Second Vendor: 126 x 1/4 = 31.5

Third Vendor: 126 x 1/3 = 42

Therefore, each vendor sold 21 towels, 31.5 towels, and 42 towels respectively.

It is worth noting that you can also find the solution by finding the least common multiple of 2,3,4 which is 12, and then multiply each ratio by 12 and get 24,36,48 respectively
