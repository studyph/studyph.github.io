17. The pile of firewood has 25 logs in the bottom row, 24 logs in the next to the bottom row, 23 logs in the third from the bottom row, and so on with one less log in each row until the top row which consists of a single log. How many logs are there in the pile


There are 325 logs in the pile.

Step-by-step explanation:

Step 1: Enumerate the given values.

Given:

a1 = 25

a2 = 24

a3 = 23

d = -1

Step 2: Identify what is being asked.

Required:

Amount of logs in the pile ( sum of the sequence)

Step 3: Solve for sum of the sequence

Solution:

The formula for the sum of the arithmetic sequence is:

S = [n (a1 + an)]/2

where S - sum of the arithmetic sequence

n - number/order of terms

a1 - first term

an - nth term, (a1, a2, a3, ... , an)

Value of n is unknown so we should solve for the value of n first by using the formula for nth term of an arithmetic series:

an = a1 + d (n - 1)

where: an - nth term, (a1, a2, a3, ... , an)

a1 - first term

d - common difference

n - number/order of terms

Substitute the value of a1, d and an

1 = 25 + [-1(n -1)]

1 = 25 - n + 1

n = 25

Find the sum

S = [25 (25 + 1)]/2

S = [(25)(26)]/2

S = 650/2

S = 325
