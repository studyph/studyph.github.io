3. what is probability histogram​


Answer:

A probability histogram is a graphical representation of a probability distribution, showing the likelihood of different outcomes on the horizontal axis and the frequency or probability on the vertical axis. It is similar to a regular histogram, but the vertical axis represents probability rather than frequency. It is used to display the probability distribution of a continuous or discrete random variable.

26. Construct the probability mass function of the discrete random variable X that has four values ( 1,2,3,4 ). Assuming that the value has an equal probability histogram and describe the pmf.​


3.1.3 Probability Mass Function (PMF)

If X is a discrete random variable then its range RX is a countable set, so, we can list the elements in RX. In other words, we can write

RX={x1,x2,x3,...}.

Note that here x1,x2,x3,... are possible values of the random variable X. While random variables are usually denoted by capital letters, to represent the numbers in the range we usually use lowercase letters such as x, x1, y, z, etc. For a discrete random variable X, we are interested in knowing the probabilities of X=xk. Note that here, the event A={X=xk} is defined as the set of outcomes s in the sample space S for which the corresponding value of X is equal to xk. In particular,

A={s∈S|X(s)=xk}.

The probabilities of events {X=xk} are formally shown by the probability mass function (pmf) of X.

In this case, the possible outcomes for X are 0, 1, and 2, which correspond to the number of red balls intwo draws from a box containing two red balls (RR), one red ball and one blue ball (RB or BR), or two blue balls (BB).
