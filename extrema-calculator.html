Answer:

f(x) = x⁴ - 4x³ + 2

Find the derivative of both sides.

f'(x) = 4x³ - 12x²

Substitute f'(x) as 0.

0 = 4x³ - 12x²

Solve for x.

x = 0

x = 3

Determine the intervals around the critical point.

⟨-∞, 0⟩, ⟨0, 3⟩

⟨0, 3⟩, ⟨3, +∞⟩

Choose a point from each interval.

x1 = -1, x2 = 1, x3 = 1, x4 = 4

Calculate the derivative of each chosen point.

f'(-1) = -16, f'(1) = -8, f'(1) = -8, f'(4) = 64

Since the derivative is negative for x < 0, and negative for 0 < x < 3, the function has no relative extrema at x = 0.

f'(1) = -8, f'(4) = 64

Since the derivative is negative for 0 < x < 3, and positive for 3 < x < +∞, the function has a relative minima at x = 3

f(x) = x⁴ - 4x³ + 2, x = 3

Substitute x as 3.

f(3) = (3)⁴ - 4(3)³ + 2

Evaluate the power.

f(3) = 81 - 4(27) + 2

Multiply.

f(3) = 81 - 108 + 2

Add and subtract.

f(3) = -25
