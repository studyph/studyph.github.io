3. Evaluate the limit of the given exponential, logarithmic and trigonometric function.1. [tex] \overset{ \text{lim}}{ \underset{ x\longrightarrow3}{}}( \frac{1}{ {3}^{2x - 5} } + 3)[/tex]​


Answer:

Hey!

[tex] \overset{ \text{lim}}{ \underset{ x\longrightarrow3}{}}( \frac{1}{ {3}^{2x - 5} } + 3) = \frac{4}{3}[/tex]

Step-by-step explanation:

Solution: Since 3 is a real number, therefore by Theorem 2 of limits (Theorem 2: Let f be a polynomial of the form) [tex] \overset{ \text{lim}}{ \underset{ x\longrightarrow3}{}} = f(3)[/tex] .

Thus,

we need to evaluate the given function [tex] \overset{ \text{lim}}{ \underset{ x\longrightarrow3}{}}( \frac{1}{ {3}^{2x - 5} } + 3)[/tex] when x = 3.

Solution:

Rewrite the given exponential function

[tex]f(3) = \frac{1}{ {3}^{2x - 5} } + 3[/tex]

Substitute x = 3 to the given function

[tex]f(3) = \frac{1}{ {3}^{2(3) - 5} } + 3[/tex]

Simplify the exponent

[tex]f(3) = \frac{1}{{3}^{1} } + 3[/tex]

Simplify

[tex]f(3) = \frac{1}{3} + 3[/tex]

[tex] f(3) = \frac{4}{3}[/tex]

Since [tex] f(3) = \frac{4}{3}[/tex] , therefore [tex] \overset{ \text{lim}}{ \underset{ x\longrightarrow3}{}}( \frac{1}{ {3}^{2x - 5} } + 3) = \frac{4}{3}[/tex].

23. 10. Logarithmic function is the____________of exponential function​


Answer:

Logarithmic functions are the inverses of exponential functions.

Explanation:

Logarithmic functions are the inverses of exponential functions. The inverse of the exponential function y = ax is x = ay. The logarithmic function y = logax is defined to be equivalent to the exponential equation x = ay. y = logax only under the following conditions: x = ay, a > 0, and a≠1. It is called the logarithmic function with base a.
