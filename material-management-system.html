What is DepEd Learning Management System

The Department of Education (DepEd) Learning Management System (LMS) is a software application or web-based technology used to plan, implement, and assess a specific learning process. The function of a learning management system is that it provides a way for instructors to create and deliver content, monitor student participation, and assess student performance. Learning management systems also serve to give students the ability to use interactive features such as threaded discussions, video conferencing, and discussion forums.

Software support for this format covers all computer platforms and handheld devices such as smartphones, tablets and more.

The advantages of PDF, and this is also the reason why PDF is so familiar to use, including:

1. PDF Format is Flexible to Use

As explained in the previous definition of PDF, where portable refers to a flexible digital product that allows this format to be opened on any device such as a computer or smartphone. Even though this format is opened on devices with different versions, it will not change the contents of the file. The contents of a PDF-extension file are fixed and will not change without the help of other software to change the contents of the text in it. This is the advantage of PDF over Microsoft Word, which tends to change the composition of the contents if opened on a different version of Microsoft.

2. PDF Has a Lock Feature

Adobe System developed a lock feature on PDF. So if you want to keep the document secret, make the document cannot be copied or want the document to be seen by others, you can use the lock feature to give a password to your PDF document.

3. Editable PDF

Worried that documents that have already been converted to PDF cannot be edited again? Or is there a chaotic arrangement of text? Don't worry, because you can edit the contents of PDF files with the help of PDF reader software.

4. PDF Can Store Images

Not only text files, in the PDF format you can also store data in the form of images, photos or graphics. In addition, converting files to PDF format can reduce the size of the file.

5. Compatible with Various Computers

Unlike the doc and other formats that can only be opened on computers with Microsoft Office software installed. The PDF format is compatible with various computer operating systems such as Windows, Linux, Mac-OS and others.
