7. There are 3 defective light bulbs in a bin containing 30 light bulbs. How many ways can you select at least 2 defective light bulbs when you choose 4 light bulbs at random?​


Answer:

We need to find the number of ways we can select at least 2 defective light bulbs out of the 4 light bulbs we choose. To do this, we can use the complement principle, which states that the number of ways to do something is equal to the total number of ways it could be done minus the number of ways it cannot be done.

There are 30 light bulbs in the bin, out of which 3 are defective, so there are 27 non-defective light bulbs. To choose 4 light bulbs at random, we have 30 total choices, out of which 27 are non-defective.

There are 30C4 = 30!/(4!26!) = 30292827/432*1 = 30,855 ways to select 4 light bulbs at random.

Out of these 30,855 ways, we need to find the number of ways that we can select less than 2 defective light bulbs.

There are 3C0 = 1 ways to select 0 defective light bulbs.

There are 3C1 = 3 ways to select 1 defective light bulbs.

[So in total there are 1+3 = 4 ways to select less than 2 defective light bulbs.]

Therefore, the number of ways to select at least 2 defective light bulbs is 30,855 - 4 = 30,851.

26. there were 9 defective bulbs in a box of 60 bulbs. what percent at the total number of bulbs were not defective? 85%​


Answer:

To find the percent of bulbs that were not defective, we need to first find the total number of non-defective bulbs in the box:

Total number of bulbs in the box = 60

Number of defective bulbs = 9

Number of non-defective bulbs = 60 - 9 = 51

Now, to find the percentage of non-defective bulbs, we can use the following formula:

Percentage = (Part / Whole) x 100

In this case, the part is the number of non-defective bulbs (51) and the whole is the total number of bulbs (60).

Percentage = (51 / 60) x 100

Percentage = 0.85 x 100

Percentage = 85%

Therefore, 85% of the total number of bulbs in the box were not defective.

29. 4. There are 15 defective products in a box containing 60 products. If 4 products are selected at random, what is the probability that 3 products are defective? ​


Answer:

0.042 or 4.2%

Step-by-step explanation:

We can solve this problem using the hypergeometric distribution, which is a probability distribution that describes the probability of k successes in n draws, without replacement, from a finite population of size N that contains exactly K successes.

In this case, we have N = 60 products in the box, K = 15 defective products, and we want to find the probability of selecting 3 defective products out of 4 without replacement.

The formula for the hypergeometric distribution is:

P(k) = (K choose k) * (N - K choose n - k) / (N choose n)

where "choose" represents the binomial coefficient.

Plugging in the values, we get:

P(3) = (15 choose 3) * (45 choose 1) / (60 choose 4)

= (455 * 45) / 487635

= 0.042
