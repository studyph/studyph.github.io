Problem

A number minus seven yields ten.

Find the number

Solution

Let x stands for a number

Key pointsminus means subtraction (-)yields means equal (=)

A number minus seven yields ten.

x - 7 = 10

x = 10 + 7, using transposition.

x = 17

Therefore, the number is 17.
