19. Anna weigh 3 kilograms more 5han her sister Nina and their total weight is 105 kilograms. what is the weight of each girl​


Answer:

The weight of Nina is 51 kg, and the weight of Anna is 54 kg.

Step-by-step explanation:

We can start solving the problem by using algebra. Let's use x as the weight of Nina and x+3 as the weight of Anna.

We know that the total weight of both girls is 105 kg, which can be expressed as:

x + (x+3) = 105

We can simplify this equation by combining like terms:

x + x + 3 = 105

2x + 3 = 105

Then we can solve for x by subtracting 3 from both sides:

2x = 102

x = 51

Therefore, the weight of Nina is 51 kg, and the weight of Anna is x + 3 = 51 + 3 = 54 kg.
