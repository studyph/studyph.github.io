One such test is the Stanford-Binet Intelligence Scales (SBIS), which assumes that IQ scores are normally distributed with a mean of 100 and a standard deviation of 16. A. Find the proportion of people in the general population with IQs between 90 to 110. • B. What is the lowest IQ score in the SBIS that would qualify you for MENSA membership? • C. Leonor has an IQ score of 150. What proportion of the general population have IQ scores above Leonor's IQ score?


Step-by-step explanation:

A. To find the proportion of people in the general population with IQs between 90 to 110, we need to standardize the IQ scores using the formula:

z = (x - μ) / σ

where:

x = the IQ score

μ = the mean IQ score (100)

σ = the standard deviation (16)

For x = 90:

z = (90 - 100) / 16 = -0.625

For x = 110:

z = (110 - 100) / 16 = 0.625

Using a standard normal distribution table or calculator, we can find the proportion of people with IQs between -0.625 and 0.625:

P(-0.625 < z < 0.625) = 0.4129 - 0.2334 = 0.1795

Therefore, about 17.95% of the general population have IQs between 90 to 110.

B. To qualify for MENSA membership, one needs to have an IQ in the top 2%. We can find the IQ score that corresponds to the top 2% by using the inverse normal distribution (z-score) table.

The z-score that corresponds to the top 2% is approximately 2.05. Therefore, we can solve for x in the following equation:

2.05 = (x - 100) / 16

x - 100 = 2.05 * 16

x = 132.8

So, the lowest IQ score in the SBIS that would qualify for MENSA membership is approximately 133.

C. To find the proportion of the general population with IQ scores above Leonor's IQ score of 150, we can again standardize the IQ score:

z = (150 - 100) / 16 = 3.125

Using a standard normal distribution table or calculator, we can find the proportion of people with IQs above 3.125:

P(z > 3.125) = 1 - 0.9992 = 0.0008

Therefore, only about 0.08% of the general population have IQ scores above Leonor's IQ
