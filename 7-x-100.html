13. 7. x² + (y + 2)² = 49 8. (x+2)² + y² = 100​


Answer:

CirclesStandard form of a circle is: (x - h)² + (y - k)² = r², where (h, k) is the vertex, and r is the radius.

7. x² + (y + 2)² = 49

(h, k) = (x - 0)² + (y + 2)² = 49(h, k) = (0, 2)

r² = 49√r² = √49r = 7

8. (x + 2)² + y² = 100

(h, k) = (x + 2)² + (y - 0)² = 100(h, k) = (2, 0)r² = 100√r² = √100r = 10


14. 5+x50+x-100+x65+x-7+x-50+x-45=360


To solve for x, we can simplify the left side of the equation by combining like terms:

5 + x + 50 + x - 100 + x 65 + x - 7 + x - 50 + x - 45 = 360

Simplifying the left side:

4x - 82 = 360

Adding 82 to both sides:

4x = 442

Dividing both sides by 4:

x = 110.5

Therefore, x = 110.5 is the solution to the equation.
