Examples:

The sum of x and y: x+y

4. What is the value of n?

- Solve for n

[tex]48 : 16 = n : 4[/tex]

[tex] \underbrace{48 : \overbrace{16 = n} : 4}[/tex]

[tex]48⋅4 = 16⋅n[/tex]

[tex] \begin{gathered} \frac{192}{16} = \frac{ \cancel{16}n}{ \cancel{16}} \end{gathered}[/tex]
