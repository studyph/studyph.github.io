11. Answers and Formula of (x+7) (x-7) = -3x


We will apply the FOIL Method. It is a mnemonic for the standard method in multiplying binomials. The step-by-step process of FOIL are the ff:

Multiply First terms of each binomialMultiply Outer terms of the binomialsMultiply the Inside terms of the binomialsMultiply the Last terms of the binomials

The products from these 4 methods will be added to each other in order to get the final term. Once the final term is obtained, calculate to get the value of x.

Step-by-step explanation:

Firsts: ( x * x)  = x²

Outsides: ( x * -7 ) = -7x

Insides: ( 7 * x ) = 7x

Last: ( 7 * -7 ) = -49

Add them up:

(x + 7) (x - 7) = -3x

x² - 7x + 7x - 49 = -3x

x² + 0 - 49 = -3x

Transpose -3x to the other side:

x² + 3x - 49 = 0

Final Answer:

The answer for (x+7)(x-7) = -3x is x² + 3x -49 = 0.
