24. Multiply (2x+5)(3x² - 2x - 4)


Answer:

6x³ + 11x² - 18x - 20

Step-by-step explanation:

(2x+5)(3x² - 2x - 4)

Distribute them first

2x(3x² - 2x - 4) + 5(3x² - 2x - 4)

Then solve

6x³ - 4x² - 8x + 15x² - 10x - 20

Combine the like terms

6x³ - 4x² + 15x² - 8x - 10x - 20

Then solve

6x³ + 11x² - 18x - 20

Answer:

= 6x³ + 11x² -18x -20

Step-by-step explanation:

(2x+5)(3x² - 2x - 4)

Distributive Property (Multiply each term in the first expression by each term in the second expression and combine like terms.)

(2x)(3x²) = 6x³

(2x)(-2x) = -4x²

(2x)(-4) = -8x

(5)(3x²) = 15x²

(5)(-2x) = -10x

(5)(-4) = -20

All terms:

6x³ -4x² -8x + 15x² -10x -20

Combine like terms.

= 6x³ + 11x² -18x -20
