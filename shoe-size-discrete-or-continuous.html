Discrete here means not interconnected (as opposed to continuous). Objects that are discussed in Discrete Mathematics - such as integers, graphs, or logical sentences - do not change continuously, but have definite and separate values. Some of the things discussed in this mathematics are settheory, combinatorial theory, number theory, permutations, functions, recursion, graph theory, and others. Discrete mathematics is the main and basic course for the field of computer science or informatics.

Discrete mathematics provides the ability to read, understand and construct mathematical arguments. As a foundation in studying computer science such as data structures, algorithms, database theory, automata and operating systems.

Continuous function is a type of function in which a continuous (continuous, uninterrupted) change in the function variable results in a continuous change in the function's output value. This means that the value of the function never changes suddenly / suddenly.

The intuitive idea of ​​continuity illustrates a continuous function as a function whose graph can be drawn without lifting the chalk from the board. More technically, a function is said to be continuous if small changes in the value of the function can be ascertained simply by making small changes to the variables.
