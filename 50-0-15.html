Answer:

d. w2-27w + 50 = 0

a=(l)(w)

p= w + w + 2w + 2w

a=50ft²

p=54ft

You need to use your information about area and perimeter to set up two equations with two variables representing length and width of the rectangle. Then you will set up a quadratic equation. Let's use L for the length and W for the width of the rectangle:

For the perimeter:

2w + 2W = 54 or

W = 27 or

W = 27 - L

For the area:

LW = 50

Now substitute W = 27 - L from the first equation into the second one:

L(27 - L) = 50 or

27w - w²= 50 or

w² - 27w + 50 = 0

This can be solved by factoring, or by using the quadratic equation.

By factoring:

(l - 2)(l - 25) = 0 or

l = 2, or l = 25

In this case, let's use 25 because length is usually the longer dimension. Now we substitute L = 25 into the first equation to find W.

W = 27 - 25 or

W = 2
