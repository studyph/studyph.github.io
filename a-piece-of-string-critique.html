2. Maria and Katy each have a piece of string. When they put the two pieces of string together end to end, the total length is 84 inches. Maria's string is 6 inches longer than Katy's. How long is Maria's piece of string? How long is Katy's piece of string?


Before solving, let's assign variables for Maria's and Katy's string.

Let m = length of Maria's string

      k = length of Katy's string

Next is to turn into equations the supporting statements of the problem.

m + k = 84  -> two pieces of strings put together end to end

k + 6 = m    -> Maria's string is 6 inches longer than Katy's

Let's use the 2nd equation to the 1st equation.

m + k = 84

(k + 6) + k = 84

2k + 6 = 84

2k = 78

k = 39 inches

Since Katy's string is 39 inches, let's use it on the 2nd formula to find the length of Maria's string:

m = k + 6

m = 39 + 6

m = 45 inches

To verify if our answer is correct, let's add up Maria and Katy's string and the answer should be equal to 84 inches:

m + k = 84

45 + 39 = 84

84 = 84

Since it is equal, the answers are:

Maria's piece of string is 45 inches.

Katy's piece of string is 39 inches.

7. 2) Cut the first string once. (a) How many pieces are there?Cut the second string twice. (b) How many pieces are there?Cut the third string thrice. (c) How many pieces are there?Cut the fourth string four times. (d) How many pieces are there?Cut the fifth string five times. (e) How many pieces are there?​


Answer:

Assuming that there are 5 strings in total

a) There will be 6 pieces when the first string is cut once

b) There will be 8 pieces when the second string is cut twice

c) There will be 11 pieces when the thrice string is cut thrice

d) There will be 15 pieces when the fourth string is cut four times

e) There will be 20 pieces when the fifth string is cut five times  

Step-by-step explanation:

We can use a function formula to get the answers. Let's solve each one using a function.

Let, f(x) be the total number of strings  

       x = initial number of strings

       n = number of strings when cut

a) f(x) = (x - 1) + n

x = 5, since we initially have 5 strings

n = 2, since we get 2 pieces when the string is cut once.

f(x) = (5 - 1) + 2

f(x) = 6 pieces of strings

b) f(x) = (x - 1) + n

x = 6, since we have 6 strings after letter a

n = 3, since we get 3 pieces when the string is cut twice.

f(x) = (6 - 1) + 3

f(x) = 8 pieces of strings

c) f(x) = (x - 1) + n

x = 8, since we have 8 strings after letter b

n = 4, since we get 4 pieces when the string is cut thrice.

f(x) = (8 - 1) + 4

f(x) = 11 pieces of strings

d) f(x) = (x - 1) + n

x = 11, since we have 11 strings after letter c

n = 5, since we get 5 pieces when the string is cut four times

f(x) = (11 - 1) + 5

f(x) = 15 pieces of strings

e) f(x) = (x - 1) + n

x = 15, since we have 15 strings after letter d

n = 6, since we get 6 pieces when the string is cut five times

f(x) = (15 - 1) + 6

f(x) = 20 pieces of strings

10. A peice of string 20 cm long, is divided into two pieces in the ratio 3:2. What is the length of each of the two pieces of string?


Answer:

12 cm and 8 cm

Step-by-step explanation:

Given:

3x

2x

Equation:

2x + 3x = 20

Solve for x:

2x + 3x = 20

5x = 20

[tex]\frac{5x}{5} = \frac{20}{5}[/tex]

x = 4

Substitute:

3x = 3(4) = 12

2x = 2(4) = 8

Answer:

12 cm and 8 cm
