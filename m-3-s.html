A. 2 m/s²

B. 3 m/s² ( answer )

C. 4 m/s²

D. 5 m/s²

Given Data:

Vi ( initial velocity ) = 26 m/s

Vf ( Final velocity ) = 47 m/s

t ( time ) = 7 seconds

a ( acceleration ) ? unknown

Solution:

a = Vf - Vi / t

  = 47 m/s - 26 m/s / 7 s

  =  3 m/s²

Answer:
