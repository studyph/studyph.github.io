25. The weights in kilos of 5 students are: 50, 55, 48, 60 and 54, Find the range


[tex]\tt{\huge{\blue{Explanation:}}}[/tex]

The range in statistics is defined as the difference between the maximum value and the minimum value. The formula for the range of ungrouped data is

[tex]\boxed{range = max \: value - min \: value}[/tex]

[tex]\tt{\huge{\red{Solution:}}}[/tex]

Based on the data shown, the maximum value is 60 and the minimum value is 48. Therefore, the range is

[tex]range = max \: value - min \: value[/tex]

[tex]range = 60 - 48[/tex]

[tex]\boxed{range = 12}[/tex]
