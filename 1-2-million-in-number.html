1. 123,400  It will still be 123,400 because the number 4 is a round down number

2. 256,000,000 It will become 260,000,000 because the number 6 is a round up number

3. 7,590,000 It will become 7,600,000 because the number 9 is a round up number

0-4 is a round down number (It won't change but the number and change the left side of the rounded number to 0)
