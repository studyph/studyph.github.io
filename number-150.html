23. Is 150 a triangular number?​


Answer:

its not a triangular number

Step-by-step explanation:

Take notice of the symmetry of the dots inside the triangle above. Each side of the triangle has the same number of dots, and one dot is added to each row of dots.

150 is a triangular number if we can arrange all 150 dots or objects in perfect symmetry to make an equilateral triangle. To determine if a number (n) is a triangular number, the following equation must be true:

√(8 × n) +1 = Integer

When we enter 150 for n in our formula, we get the following result:

√(8 × 150) +1 ≈ 34.6554

34.6554 is not an integer (whole number), therefore the answer to "Is 150 a triangular number?"
