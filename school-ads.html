In adding polynomials, you should apply the same as adding integers...same signs add different signs subtract, but in polynomial you should just add the polynomials with the same exponent. Same as subtracting polynomials, you should also apply the rules in subtracting integers, change the sign of the subtrahend and proceed to addition. Again, just subtract polynomials with the same degree.
