19. what is the color of Balack cloth if it was hit by the light?,why?​


Answer:

When light hits an object made of matter, each photon is either absorbed, reflected, or refracted. If the light is reflected, the object will not appear black. If the light is refracted, the object will appear translucent or transparent. In an object that appears black, the photons are absorbed.
