Let u = cos(x)

Then the equation becomes:

cos(2x) + cos(x) + 1 = 0

=> 2cos^2(x) - 1 + cos(x) + 1 = 0

=> 2u^2 + u - 1 = 0

We can solve this quadratic equation using the quadratic formula:

u = (-b ± sqrt(b^2 - 4ac)) / 2a

where a = 2, b = 1, and c = -1

u = (-1 ± sqrt(1^2 - 4(2)(-1))) / 2(2)

u = (-1 ± sqrt(9)) / 4

u1 = -1/2

u2 = 1/2

Now, we need to solve for x using the solutions we found for u:

When u = -1/2:

cos(x) = -1/2

x = 2π/3 or 4π/3

When u = 1/2:

cos(x) = 1/2

x = π/3 or 5π/3

Therefore, the solutions of the equation cos 2x + cos(x)+1 = 0 in the interval [0, 2π) are:

x = π/3, 2π/3, 4π/3, 5π/3.

(sin x + cos x)² = 1 + 2sinxcosx
by expanding LHS,
sin²x + 2sinxcosx + cos²x = 1 + 2sinxcosx
By the Pythagorean Identity sin²x + cos²x = 1 and substituting it to the LHS, we get:

1 + 2sinxcosx = 1 + 2sinxcosx

Thus, proving the identity.

Y' = -csc^2(3x^2 - x) * (6x - 1)

Thus, the derivative of Y = cot(3x^2 - x) is Y' = -csc^2(3x^2 - x) * (6x - 1).

Y' = -e * sinx
