7. Integrate x(sinx)^4 cos4x dxPlease show the complete solution. Thank you!​


To solve this integral, we can use the following formula:

∫ u^n v^m dx = (1/m+1) * u^(n+1) * v^(m+1) + C

where 'u' and 'v' are functions of 'x', 'n' and 'm' are constants.

Applying this formula to our integral, we can 'let u = sinx' and 'v = cosx'. Then 'n = 4' and 'm = 4'.

Substituting these values into the formula, we get:

∫ (sinx)^4 (cosx)^4 dx = (1/(4+1)) * (sinx)^5 * (cosx)^5 + C

= (1/5) * (sinx)^5 * (cosx)^5 + C

so the solution to the integral is:

∫ x(sinx)^4 cos4x dx = (1/5) * ∫ (sinx)^5 * (cosx)^5 dx + C
