Using Gauss theorem to find the sum of the first nth integers

   S₂₀₀      =   n(n+ 1)/2

                =   200(201)/2

                =      20100

   thus the sum of the next 100 integers

                  =     20100 - 5050

                  =15,050

16. Compute the sum of the first 100 positive integers exactly dividible by 7


Hello! We can easily solve this problem using an arithmetic sequence :>


Just to recall, the formula for the sum of an arithmetic sequence is:

Sn = (a1 + an)(n/2); wherein a1 is the first term, an is the last term, and n is the number of terms.


So first, we need 100 positive integers exactly divisible by 7, which would mean our difference is 7, and our n = 100. The first term is automatically 7 too. Solve for the 100th term:

a100 = a1 + (n-1)(d)

a100 = 7 + (99)(7)

a100 = 700


Now, we can solve for the sum.

Sn = (a1 + an)(n/2)

S100 = (a1 + a100)(100/2)

S100 = (7 + 700)(50)

S100 = 35350


The sum of the first 100 positive integers exactly divisible by 7 is 35, 350.
