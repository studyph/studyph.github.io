Problem: Find the roots of the quadratic equation x² + 5x + 24 = 0 by using the quadratic formula.

[tex]a = 1 \: ; \: b = 5 \: ; \: c = 24[/tex]

- Find the solution of the quadratic equation using the quadratic formula.

[tex] \begin{aligned}& \bold{ \color{lightblue}Formula:} \\& \boxed{x = \frac{ \text-b ± \sqrt{ {b}^{2} - 4ac } }{2a}} \end{aligned}[/tex]

[tex] \begin{aligned}{x = \frac{ \text-5 ± \sqrt{ {5}^{2} - 4(1)(24) } }{2(1)}} \end{aligned}[/tex]

[tex] \begin{aligned}{x = \frac{ \text-5 ± \sqrt{25 - 4(1)(24) } }{2(1)}} \end{aligned}[/tex]

[tex] \begin{aligned}{x = \frac{ \text-5 ± \sqrt{25 - 4(24) } }{2}} \end{aligned}[/tex]

[tex] \begin{aligned}{x = \frac{ \text-5 ± \sqrt{25 - 96 } }{2}} \end{aligned}[/tex]

[tex] \begin{aligned}{x = \frac{ \text-5 ± \sqrt{ \text -71} }{2}} \end{aligned}[/tex]

Root #1

[tex] \begin{aligned}{x = \frac{ \text-5 + \sqrt{ \text -71} }{2}} \end{aligned}[/tex]

Root #2

[tex] \begin{aligned}{x = \frac{ \text-5 - \sqrt{ \text -71} }{2}} \end{aligned}[/tex]

- Therefore, the roots of the quadratic equation are:

[tex] \large \rm Roots =\small\boxed{ \rm \green{ \: \frac{ \text - 5 + \sqrt{ \text - 71} }{2} , \frac{ \text - 5 - \sqrt{ \text - 71} }{2} \: }}[/tex]===============================
