12. 5x + 2y = 12 5x – 2y = 28


To solve for x and y in the system of equations:

5x + 2y = 12 (Equation 1)

5x - 2y = 28 (Equation 2)

We can use the elimination method by adding the two equations to eliminate y:

5x + 2y + 5x - 2y = 12 + 28

10x = 40

Then, we can solve for x by dividing both sides by 10:

x = 4

Next, we can substitute x = 4 into one of the original equations (say, Equation 1) to solve for y:

5x + 2y = 12

5(4) + 2y = 12

20 + 2y = 12

2y = -8

y = -4

Therefore, the solution to the system of equations is:

x = 4

y = -4

So, A = 4 and B = -4. Note that there are no variables in the set of English alphabet letters, so A and B are simply arbitrary symbols we use to represent the solutions for x and y, respectively.
