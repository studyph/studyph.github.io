If I had an 80GB hard disk, I would partition it as follows:

30GB primary partition for my OS and software applications

20GB for multimedia files (photos, videos, etc.)

10GB for personal documents (essays, resumes, etc.)

10GB for work-related documents (spreadsheets, presentations, etc.)

5GB for miscellaneous files (downloads, temporary files, etc.)

5GB for backup/archival files (system backups, archives, etc.)

This partition scheme allows me to keep my files organized and easily accessible, while also providing a dedicated space for backups and archives.

Your best option: do a full disk-image backup on a regular basis, with data included, using software that can read images and selectively pull files for restoration when necessary.
