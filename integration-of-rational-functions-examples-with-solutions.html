15. any examples for integration of exponential function?​


Answer:

Yes, here are some examples of integration problems involving exponential functions:

∫ e^x dx = e^x + C, where C is the constant of integration.

∫ 3e^2x dx = (3/2)e^2x + C, where C is the constant of integration.

∫ e^(-x) dx = -e^(-x) + C, where C is the constant of integration.

∫ 2e^(-2x) dx = -e^(-2x) + C, where C is the constant of integration.

∫ e^(2x) dx = (1/2)e^(2x) + C, where C is the constant of integration.

∫ 5e^(3x) dx = (5/3)e^(3x) + C, where C is the constant of integration.

Note that all these integrals involve only the exponential function and can be evaluated using the power rule of integration.
