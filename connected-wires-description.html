30. 1. Give a description on how parts are being connected in a series circuit. 2. What do you think will happen if one of the bulbs in series is damaged (open filament)? 3. What do you think is the reason why series connection is not used as standard for house and building wiring​


Answer:

1. connected along a single "electrical path" and each component has the same current through it

2.in a series circuit, every device must function for the circuit to be complete. One bulb burning out in a series circuit breaks the circuit. In parallel circuits, each light has its own circuit, so all but one light could be burned out, and the last one will still function

3.Series arrangements are not used in household circuits, because even if one of the elements get short-circuited, the complete circuit will break while in the parallel circuit, if one arm of the circuit gets broken, all the other arms will still work independent of that.
