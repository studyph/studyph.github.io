3/1 = x/9

Where x represents the number of roses that can be put in 9 vases.

To solve for x, we can cross-multiply and simplify:

3(9) = 1(x)

27 = x

So Anita can put 27 roses in 9 vases.

To find how many roses can be put in 3 vases, we can set up another proportion:

Where x represents the number of roses that can be put in 3 vases.

3(3) = 1(x)

9 = x

So Anita can put 9 roses in 3 vases.

Finally, to find how many roses can be put in 1 vase, we can set up another proportion:
