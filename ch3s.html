16. identify what is wrong with each of the following structures . Encircle your answer . 1. CH3-CH3-(aboveCH3) CH -CH3 2. CH3 (above CH3) CH ( NASA baba CH3 ) CH3 . 3. CH3 CH2 CH2 (nasababa CH3) CH2 CH3 4. CH3 CH2 CH2 CH (nasababa CH3 CH2) CH2 (NASA ta@s CH3) CH2 CH3 5. CH3 CH2 (nasa baba CH2 ) CH2. 6. CH2 CH (nasababa CH3) CH =CH (nasababa CH3) CH3 ​


Answer:

Note that four bonds can only be formed by carbon atoms. The incorrect ones are in bold.

1. CH3-CH3-(aboveCH3) CH -CH3

There should only be 2 hydrogen atoms

2. CH3-(above CH3) CH-( NASA baba CH3 ) CH3.

It should be CH2.

3. CH3 CH2 CH2-(nasababa CH3) CH2-CH3

There should only be one hydrogen atom (CH) because carbon atoms only have four bonds and the other one is bonded to methyl (CH3)

4. CH3-CH2-CH2-CH-(nasababa CH3 CH2) CH2- (NASA ta@s CH3) CH2-CH3

1st CH should be CH2, second, it should be CH2CH3, third and fourth should be CH.

5. CH3-CH2-(nasa baba CH2 ) CH2.

Should be CH3

6. CH2-CH-(nasababa CH3) CH =CH (nasababa CH3)-CH3

1st should be CH3- CH2, they both have one hydrogen atom missing to complete the 4 carbon bonds. 2nd, CH= CH should only be C=C, no hydrogen atom bond because the double bond is attached to the carbon atom.

P.S. Hindi nakalagay nang dash sa ibang number kaya ako nalang yung naglagay. Oakicheck lang po if sakto po yung pagkalagay ko sa mga dash/ bond each number.
