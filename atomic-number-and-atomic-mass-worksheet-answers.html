Most Essential Learning Competeney: Determine the number of protons, neutrons, and electrons K to 12 BEC CG: SSMT-111e-f-10 In a particular atom. Atomic Number-Represents both the number of protons and electrons in an element Most of the mass comes from the neutron and proton. To find the number of neutrons subtract the atomic weight from the atomic mass 2 He 4 Score: Atomic Symbol- a short hand way of writing each elements name Atomic Mass-The weight of an element​


The Protons are the same the Atomic number so in the case of He its 2. The Neutrons are found by subtracting the Atomic Number from the Atomic Mass. For example: 4 - 2 = 2 So for He there would be 2 protons and 2 neutrons. The Electrons are equal to the Protons, so for He there would be 2 electrons.

.

Identify the atomic symbol for the element you are studying. This is a shorthand way of writing the element's name. For example, "He" represents the element helium.Determine the atomic mass of the element. This is the weight of the element, which includes both protons and neutrons. For example, the atomic mass of helium (He) is 4.Use the atomic number to determine the number of protons and electrons in the atom. The atomic number is equal to the number of protons in the nucleus of an atom, which also determines the number of electrons in the electron cloud surrounding the nucleus. For example, the atomic number of helium (He) is 2, so it has 2 protons and 2 electrons.Subtract the atomic number from the atomic mass to determine the number of neutrons in the atom. For example, the number of neutrons in helium (He) is 4 - 2 = 2.

Therefore, the full P.E.N. for helium (He) is:

Protons: 2

Electrons: 2

Neutrons: 2

Answer:

To calculate the numbers of subatomic particles in an atom, use its atomic number and mass number: number of protons = atomic number. number of electrons = atomic number.

How many neutrons does Mercury 201 have?

We see that for mercury the atomic number is 80. The atomic number is the number of protons in an atom's nucleus, so we can tell right away that an atom of mercury contains 80 protons. The mass number (201) is the total number of protons and neutrons. So, there must be 201 - 80 = 121 neutrons.\
