RATE PERCENTAGE

1. 25%. 14

2. 20%. 18

3. 45%. 162

4. 60%. 48

5. 12%. 15

Solution:

rate x base = percentage

Let n = base

1.) 25% n = 14

(25/100)n = 14

n = 14(100)/25

n = 56

2.) 20% n = 18

(20/100)n = 18

n = 18(100)/20

n = 90

3.) 45% n = 162

(45/100)n = 162

n = 162(100)/45

n = 360

4. 60% n = 48

(60/100)n = 48

n = 48(100)/60

n = 80
