=> 8x² - 8x + 32 = 80

=> 8x² - 8x - 48 = 0

=> x² - x - 6 = 0 [Dividing by 8 throughout]

=> x² + 2x - 3x - 6 = 0

=> (x² + 2x) - (3x + 6) = 0

=> x(x + 2) - 3(x + 2) = 0

=> (x + 2)(x - 3) = 0

=> (i) x + 2 = 0, or (ii) x - 3 = 0
