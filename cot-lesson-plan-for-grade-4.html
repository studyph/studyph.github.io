22. prove (1-cot^2x/1+cot^2x) = sin^2x - cos^2x​


We can start with the left-hand side of the equation and use the identity for cotangent:

(1 - cot^2x) / (1 + cot^2x)

= [(1 - cos^2x/sin^2x) / (1 + cos^2x/sin^2x)] (since cot^2x = cos^2x/sin^2x)

= [(sin^2x - cos^2x) / sin^2x * (1 + cos^2x/sin^2x)] (multiplying numerator and denominator by sin^2x)

= [(sin^2x - cos^2x) / (sin^2x + cos^2x)] * [(sin^2x + cos^2x) / sin^2x]

= (sin^2x - cos^2x) / sin^2x

= sin^2x / sin^2x - cos^2x / sin^2x

= 1 - cos^2x / sin^2x

= sin^2x - cos^2x

Therefore, we have shown that (1 - cot^2x) / (1 + cot^2x) = sin^2x - cos^2x.

cot A - cos A = (1 - sin A) * (cot A + cos A) / (1 + sin A)

Expanding the right-hand side and canceling the common terms, we get:

cot A - cos A = cot A * cos A / (1 + sin A)

Dividing both sides by cos A, we get:

cot A = cot A * cot A / (1 + sin A)

Isolating cot A on one side, we get:

cot A * (1 + sin A) = cot^2 A

Dividing both sides by (1 + sin A), we get:

cot A = cot^2 A

30. Find the fundamentals identity of[tex] \frac{tan \: x \: + \: cot \: x}{cot \: x} [/tex]​


SOLUTION

In order to find the Fundamental Identity given trigometric expression, first we need to express it with [tex] cos [/tex] and [tex] sin [/tex]. Then, we use Pythagorean and Trigometric Identity.

[tex] \rm{\frac{tan \: x \: + \: cot \: x}{cot \: x}} \\\\ \rm{= \frac{\frac{cos \ x}{sin \ x} + \frac{sin \ x}{cos \ x}}{\frac{cos \ x}{sin \ x}}} \\\\ \rm{= \frac{\left(\frac{cos \ x}{sin \ x}+\frac{sin \ x}{cos \ x}\right)sin \ x}{cos \ x}} [/tex]

Join the two fractions in numerator

[tex] \rm{= \frac{\frac{cos^2 \ x + sin^2 \ x}{sin \ x \ cos \ x} sin \ x}{cos \ x}} \\\\ \rm{= \frac{cos^2 \ x + sin^2 \ x}{cos \ x \ cos \ x}} \\\\ \rm{= \frac{cos^2 \ x + sin^2 \ x}{cos^2 \ x}} \\\\ \rm{= \frac{1}{cos^2 \ x}} \\\\ \boxed{\rm{= sec^2 \ x}} [/tex]
