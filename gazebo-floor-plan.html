2. 2. GAZEBO. The floor of a gazebo is in the shape of aheptagon. Four of the interior angles measure 135º. Theother interior angles have equal measures. Find theirmeasures.plsss help me solve this problemneed now​


Answer:

formula for sum of interior angles is:

sum of interior angles = (n-2)*180

where n = # of sides of the polygon

for heptagon, it has 7 sides so n =7

so let's compute for sum of interior angles when n=7.

sum of interior angles = (n-2)*180

sum of interior angles = (7-2)*180

sum of interior angles = (5)*180

sum of interior angles = 900°

we will use this knowledge later...

Now we can compute the 7 interior angles of the heptagon.

We already know the first four:

∠1 = ∠2 = ∠3 = ∠4 = 135°

and we also know that the other three are also equal, but we don't know its measurement yet, so  

∠5 = ∠6 = ∠7 = ?

Since each angles of 5, 6, and 7 have the same measurement, let's assign all of them a single letter called x.

So we can now say:

∠5 = x

∠6 = x

∠7 = x

Now let's solve for x:

This is why we computed the sum of interior angles earlier,

sum of interior angles = ∠1 + ∠2 + ∠3 + ∠4 + ∠5 + ∠6 + ∠7  

where,

sum of interior angles = 900°

∠1 = ∠2 = ∠3 = ∠4 = 135°

∠5 = ∠6 = ∠7 = x

so,

sum of interior angles = ∠1 + ∠2 + ∠3 + ∠4 + ∠5 + ∠6 + ∠7  

900 = 135 + 135 + 135 + 135 + x + x + x  

900 = 540 + 3x

3x = 900 - 540

3x = 360

x = 360/3

x = 120°

Therefore the remaining 3 interior angles each measure 120°.

where n is the number of sides of the polygon.

Interior angle = 180 - (360 / 8)

Interior angle = 180 - 45

Interior angle = 135 degrees

Therefore, the angle at which the lumber should be cut to frame the base of the gazebo is 135 degrees.

To fit the base of the gazebo, the carpenter needs to cut eight pieces of wood at an angle that matches the sides of the regular octagon.

A regular octagon has eight sides of equal length and eight angles of equal size. To find out what angle to cut the wood at, we need to find the measure of one of these angles.

The formula to find the measure of each angle in a regular octagon is:

angle measure = (n-2) x 180 / n

where n is the number of sides of the polygon.

Substituting n=8 in the formula, we get:

angle measure = (8-2) x 180 / 8
angle measure = 6 x 180 / 8
angle measure = 135 degrees

Therefore, the carpenter should cut the wood at a 135 degree angle to frame the base of the gazebo.
