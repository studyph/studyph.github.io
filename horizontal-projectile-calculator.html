2. ACTIVITY Direction: Given the angle and initial velocity, calculate the horizontal and vertical components of projectile.​


Explanation:

The formulas we are going to use to compute for the horizontal and vertical components of initial velocity are

vix = vi cos θ

and

viy = vi sin θ

where

vix = horizontal component of velocity

viy = vertical component of velocity

vi = initial velocity

θ = angle of release (from the horizontal)

-----------------------------------------------------------

Solution: (30° and 10 m/s)

For the horizontal component

vix = vi cos θ

vix = (10 m/s)(cos 30°)

vix = (10 m/s)(0.866)

vix = 8.66 m/s

For the vertical component

viy = vi sin θ

viy = (10 m/s)(sin 30°)

viy = (10 m/s)(0.5)

viy = 5 m/s

-----------------------------------------------------------

18. The initial horizontal speed of this projectile is 11 m/s. The initial vertical speed is 9 m/s. a.) What is the horizontal speed of this projectile at point 2? b.) What is the horizontal speed of this projectile at point 4? c.) What is the horizontal speed of this projectile at point 5? d.) What is the horizontal speed of this projectile at point 6? e.) What is the horizontal speed of this projectile (refer to the figure in item 5) at point 3? f.) What is the vertical speed of this projectile (refer to the figure in item 5) at point 3 (it's maximum height)?With formula and solution ​


Answer:

To solve these problems, we can use the following kinematic equations for projectile motion:

Vertical motion:

vf = vi + at

y = vi*t + (1/2)at^2

vf^2 = vi^2 + 2ay

Horizontal motion:

x = v*t

Where:

vf = final velocity

vi = initial velocity

a = acceleration due to gravity (-9.8 m/s^2)

t = time

y = vertical displacement

x = horizontal displacement

v = velocity

Given:

vi_x = 11 m/s (initial horizontal speed)

vi_y = 9 m/s (initial vertical speed)

a) At point 2, the horizontal speed is still 11 m/s since there is no acceleration in the horizontal direction.

b) At point 4, the projectile has the same horizontal speed of 11 m/s as there is no horizontal acceleration.

c) At point 5, the projectile is moving horizontally and there is no acceleration, so the horizontal speed is still 11 m/s.

d) At point 6, the projectile is moving horizontally and there is no acceleration, so the horizontal speed is still 11 m/s.

e) At point 3, the projectile has reached its maximum height and the vertical velocity is 0 m/s. Therefore, the horizontal speed is still 11 m/s.

f) At point 3, the vertical speed is 0 m/s because the projectile has reached its maximum height. At this point, the vertical velocity changes direction and begins to decrease as the projectile falls back down. Using the equation vf^2 = vi^2 + 2ay, we can find the initial vertical velocity:

0^2 = 9^2 + 2*(-9.8)*y

y = 4.6 m

Now, we can use the equation vf = vi + at to find the final vertical velocity at point 3:

0 = 9 + (-9.8)*t

t = 0.92 s

vf = vi + at

vf = 9 + (-9.8)*0.92

vf = -0.16 m/s

Therefore, the vertical speed at point 3 is -0.16 m/s (moving downwards).
