Find the solution set of 16x +24 = 7x + 60 Consider the replacement set {-2, 4, 9}, whose values or elements are arbitrarily given.

Replace the x with -2, then 4, and 9 to the equation of 16x +24 = 7x + 60.

8x + 16 = 72 8(7) + 16 = 72 56 + 16 = 72 72 = 7272 = 72 is True

Substitute the equation if x = 3.

8x + 16 = 72 8(3) + 16 = 72 24 + 16 = 7240 ≠ 7240 = 72 is False

Find the solution set of 16x +24 = 7x + 60 Consider the replacement set {-2, 4, 9}, whose values or elements are arbitrarily given.

SOLUTION:

If x = -2, then

16x +24 = 7x + 6016(-2) +24 = 7(-2) + 60-32 + 24 = -14 + 60-8 = 46 ✖

if x = 4.

16x +24 = 7x + 6016(4) +24 = 7(4) + 6064 + 24 = 28 + 6088 = 88 ✔

if x = 9.

16x +24 = 7x + 6016(9) +24 = 7(9) + 60144 + 24 = 63 + 60168 = 123 ✖

ANSWER:

Therefore the solution of set is 4.
