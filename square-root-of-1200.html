7. If 1200 cm2 of material is available to make a box with a square base and an open top, find the largest possible volume of the box.


Word Problem Involving Finding the Volume of the Box

Problem:

Find the box's maximum volume if 1200 cm2 of material is available to create one. The box will have a square base and an open top.

Solution:

Step1:  Let [tex]x[/tex] > [tex]0[/tex]  be the length of a side of the base of the box. Let  [tex]y[/tex] > [tex]0[/tex] be the height of the box.

Therefore, the surface of the box is [tex]x^{2} + 4[/tex] · [tex]x[/tex] · [tex]y[/tex].

(see attachment below)

Step 2: Since we have 1200cm² of material, it is derived as:

[tex]x^{2} + 4xy = 1200[/tex]

Therefore,

[tex]y = \frac{1200 - x^{2} }{4x}[/tex]

Step 3: The box has a volume of

[tex]V = x^{2}[/tex] · [tex]y[/tex]

Consequently, we can write it as a function of [tex]x[/tex] because of (1):

[tex]V(x) = x^{2}[/tex] · [tex]\frac{1200 - x^{2} }{4x} = \frac{1}{4} (1200x-x^{3} )[/tex]

Step 4: Finding the derivative of [tex]V[/tex] is necessary because we want the volume to be as large as possible.

[tex]V'(x) = \frac{1}{4} (1200 - 3x^{3} ).[/tex]

Step 5: Observe that [tex]V'[/tex] is set forth for each [tex]x[/tex] . Locate the zeros:

[tex]V' (x) = 0[/tex] ⇔ [tex]1200 - 3x^{3} = 0[/tex]

               ⇔ [tex]x^{2} = 400[/tex]

               ⇔ [tex]x = 20[/tex]

Step 6:

Note that,

[tex]V' (x) < 0 for all x > 20[/tex]

and

[tex]V' (x) < 0 for all x > 20[/tex]

Therefore,  [tex]V(20) = \frac{1}{2} (1200[/tex] · [tex]20 - 20x^{3} ) = 4000[/tex]

4000 is the volume function's absolute maximum value.
