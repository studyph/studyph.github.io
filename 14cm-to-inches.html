1. find the circumference of a circler=14cmd=20cmr=15cmr=9 inchesd=21 inchesr =means radiusd=diameterpasagot po pls need now with solutions po pls​


Circumference of a circle can be calculated using the formula:

C = 2πr

Where:

C = Circumference of the circle

π = Pi, approximated to 3.14

r = Radius of the circle

Using the given values:

For r = 14 cm,

C = 2 x 3.14 x 14

C = 87.92 cm

For d = 20 cm (which is equivalent to 2r),

r = d/2 = 20/2 = 10 cm

C = 2 x 3.14 x 10

C = 62.8 cm

For r = 15 cm,

C = 2 x 3.14 x 15

C = 94.2 cm

For r = 9 inches,

C = 2 x 3.14 x 9

C = 56.52 inches

For d = 21 inches (which is equivalent to 2r),

r = d/2 = 21/2 = 10.5 inches

C = 2 x 3.14 x 10.5

C = 65.94 inches

Therefore, the circumference of the circle is 87.92 cm, 62.8 cm, 94.2 cm, 56.52 inches, and 65.94 inches for r=14cm, d=20cm, r=15cm, r=9 inches, and d=21 inches, respectively.
