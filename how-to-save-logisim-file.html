19. You have an 8 GB flash drive. You want to save a file with a size of 120MB. Howmany 120MB files you can store in your flash drive?Show your solution here:​


Answer:

You can store 120 MB in 8 GB flash drive 66 times. Leaving a free space of 80 MB. If we won't leave a free space, you can store the 120 GB 66.667 times instead,

Explanation:

There are 1000 MB per 1 GB in decimal conversion.

There are 1024 MB per 1 GB in binary conversion.

Supposing that you are counting naturally and using decimal conversion:

In an 8 GB flash drive, 8000 MB files can be stored.

If we will divide 120 MB from 8000 MB, the answer is going to be of floating data type. We don't want that.

So we will use an operator called modulo. The symbol for modulo operator is %. In our formula, suppose that a is the number of free space you will leave if you store 120 MB file in an 8 GB flash drive until you can't add another file with the same size.

a = 8000%120

a = 80

To get how many exactly 120 MB we can store, we can subtract a from 8000 MB then divide by the size of the file. Suppose that b is the exact number of times you can store 120 MB in an 8000 MB flash drive.

b = (8000-a)/120

b = (8000-80)/120

b = 7920/120

b = 66

We can store 120 MB in an 8 GB flash drive 66 times. Leaving a free space of 80 MB.

The overall solution to this would be:

a = [(storage)-((storage)%(file size))]/(file size)

a = [(8*1000)-[(8*1000)%120]]/120

a = [8000-(8000%120)]/120

a = [8000-80]/120

a = 7920/120

a = 66

8 is multiplied by 1000 because 8 is GB and the file size is written in MB. We converted GB to MB to have a common unit.
