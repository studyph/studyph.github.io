18. The Presentation app included in Google Drive is called? a. Google Docs b. Google Sheets c. Prezi d. Google Slides


The answer is letter D. Google Slides .

Google Slides is a web based software, and a free presentation program offered by Google. It allows user  to create and edit presentations online.

Google Docs is a web-based software office suite offered by Google within its Google Drive service. It is a free word processor program. It allows users to create and edit files online. It is available as a web application, and also as mobile app. The app is compatible with Microsoft Office file formats. Service also includes Google Sheets and Google Slides.

Google Sheets is a web-based program offered by Google within its Google Drive service. It is also available as a web application, and mobile app. It is a free spreadsheet program. It allows users to create and edit files online.

Prezi is a short term for “presentation” in Hungarian. It is a presentation software company founded in 2009. Presentations features a map-like overview, unlike a traditional slide based presentation. It allows users to pan between topics, and zoom in on details, and pull back to reveal context.
