21. find x if me=2x+1 and no=3x-48 × 3x-45 × 2x+1​


Answer:

We are given two equations:

me = 2x + 1

no = 3x - 4

We are asked to find x in the expression:

(8 × 3x-4) / (5 × 2x+1)

We can start by simplifying this expression:

(8 × 3x-4) / (5 × 2x+1) = (24x-32) / (10x+5)

Now we can substitute the values of me and no into this expression:

(24x-32) / (10x+5) = (me × no) / 2

Substituting me = 2x+1 and no = 3x-4, we get:

(24x-32) / (10x+5) = (2x+1) × (3x-4) / 2

Multiplying both sides by 2(10x+5), we get:

2(24x-32) = (10x+5)(2x+1)(3x-4)

Expanding the right side, we get:

2(24x-32) = (60x^2 - 70x - 5)

Simplifying both sides, we get:

48x - 64 = 30x^2 - 35x - 2

Rearranging and simplifying, we get:

30x^2 - 83x + 62 = 0

We can solve for x using the quadratic formula:

x = [83 ± sqrt(83^2 - 4×30×62)] / (2×30)

x = [83 ± sqrt(5041)] / 60

x = [83 ± 71] / 60

x = 2/3 or x = 31/30

Therefore, x can have two possible values: x = 2/3 or x = 31/30.
