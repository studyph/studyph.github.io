26. QUADRATIC FUNCTION y=3(x-2)²+5STEPS 1.) expand (x+2)²2.) multiply the perfect square trinomial by 3 3.) add 54.simplify5. result​


QUADRATIC EQUATION

Steps:

1. Expand (x-2)²

[tex]\small{\purple{\boxed{\pink{x^2 - 4x + 4}}}}[/tex]

(The quantity of (x-2) must be squared, so that it can be changed into its expanded form)

2. Multiply the perfect square trinomial by 3

[tex]\small{\pink{\boxed{\purple{y = 3(x^2 - 4x + 4)}}}}\\\small{\pink{\boxed{\purple{y = 3x^2 - 12x + 12}}}}[/tex]

3. Add 5

[tex]\small{\purple{\boxed{\pink{y = 3x^2 - 12x + 12 + 5}}}}\\\small{\purple{\boxed{\pink{y = 3x^2 - 12x + 17}}}}[/tex]

4. Simplify

[tex]\small{\pink{\boxed{\purple{y = 3 x 2 - 12x + 17}}}}[/tex]

(Step no. 3 is just the same to step no. 4, no need to simplify them)

5. Result:

[tex]\small{\pink{\boxed{\purple{y = 3 x 2 - 12x + 17}}}}[/tex]
