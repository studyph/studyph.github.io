1. 1. The Mathematics Society wants to construct a rectangular billboard in their school with an areaof 275 m2. The length of the billboard is 14 m more than its width. What are the dimensions of thebillboard? Write your SOLUTIONS below.​


Answer:

length = 25 m

width = 11 m

Step-by-step explanation:

Area = 275 m²

length = 14 m more than its width = w + 14

Let:

A = Area

l = Length (lowercase L)

w = width

SOLUTION:

FORMULA OF AREA OF A RECTANGLE:

l × w = 275

if l = w+ 14

(w + 14)(w) = 275

w² + 14w = 275

By Completing the square:

w² + 14w +(14/(2×1))² = 275 +(14/(2×1))²

w² + 14w + (14/2)² = 275 + (14/2)²

w² + 14w + (7)² = 275 + (7)²

w² + 14w + 49 = 275 + 49

(w + 7)² = 324

√(w + 7)² = √324

w + 7 = ±18

For roots:

w + 7 = 18

w + 7 - 7 = 18 - 7

w = 18 - 7

w = 11 ANSWER

w + 7 = -18

w + 7 - 7 = -18 - 7

w = -18 - 7

w = -25

ROOTS: 11 and -25

FACTORS: (x-11)(x+25)

Since we are talking about dimensions, neglect the negative root. Substitute the positive root to the original formula of area.

w = 11 m

A = 275 m²

length × width = Area

l × w = A

l × 11 m = 275 m²

(l × 11 m)/11 m = 275 m²/11 m

l = 275 m²/11 m

l = 25 m ANSWER

Answer:

Answer:

length = 21 ft.

Step-by-step explanation:

let l = length

let w = width

Given:

l = 3 ft. more than thrice the width

l = 3w + 3

area = 126 ft²

Find the length

area = length x width

area = lw

126 = (3w + 3)w

126 = 3w² + 3w

3w² + 3w = 126

3w² + 3w - 126 = 0

Factor the expression

3(w² + w - 42) = 0

w² + w - 42 = 0

w² + 7w - 6w - 42 = 0

w(w + 7) - 6(w + 7) = 0

(w + 7)(w - 6) = 0

We cannot use the factor (w + 7) because it's negative

w - 6 = 0

w = 6

Therefore,

width = 6  ft.

length = 3(6) + 3 = 21 ft.
