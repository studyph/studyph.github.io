30. The realized return on a stock portfolio is the weighted average of the expected returns on the stocks in the portfolio.


False, the realized return on a stock portofolio is the weighted average of the expected returns on the stocks portofolio.

Explanation :

The realized return on a stock portfolio and the expected return on a stock portfolio are two different things that are not interrelated. The difference between the two lies in the time at which transactions in one's portfolio take place.

Realized returns refer to returns achieved in the past, and expected returns refer to returns that are anticipated over future periods. The required return is the minimum expected rate of return an investor would need to invest in an asset over a given period, given the risk of the asset.
