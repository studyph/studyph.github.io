2. Definition of Random Access Memory (RAM)​


Definition of Random Access Memory (RAM)​

RAM (random access memory) is a sort of computer memory that may be read and changed in any order and is widely used to store working data and machine code. Unlike other direct-access data storage media (such as hard disks, CD-RWs, DVD-RWs, and older magnetic tapes and drum memory), where the time needed to read and write data items varies wildly relying on their physical places on the recording medium due to computerized restrictions, a random-access memory device authorizes data items to be read or written in nearly the same amount of time regardless of their physical location within the memory.

To link the data lines to the designated store for reading or writing the entry, RAM contains multiplexing and demultiplexing hardware. The same address usually accesses more than one bit of storage, thus RAM devices frequently have many data lines and are referred to as "8-bit," "16-bit," and so on.
