9. colors reflected by black the color of an object that al​


Answer:

Black objects absorb all colours so no light is reflected.

Explanation:

When an object appears to be black, this means that it is not reflecting any colors, but is instead absorbing them all. The visible light spectrum, or white light, is made up of many different wavelengths of light that each represents a color. The colors of the visible spectrum are red, orange, yellow, green, blue, indigo, and violet. These different wavelengths either reflect off an object or are absorbed by the object. The wavelengths that reflect represent the colors we see in an object. For example, a red shirt would absorb all the other colors in white light but reflect red. Black isn't technically a color, since it absorbs all colors. White, by contrast, reflects all colors and therefore doesn't show up as a color either.

1.many objects have color because they contain pigments. a pigment is a substance that colors materials by reflecting light of certain wavelengths and absorbing light of other wavelengths. a very common pigment is the dark green pigment called chlorophyll, which is found in plants.The color of an object depends on two factors 2.– the color of the light source and which wavelengths of light the object reflects. To view the true color of an object, we must illuminate it with a light source that includes all the wavelengths that the object will reflect3.Yes, color as we see it is only a perception - it is created within the visual system as a response to stimulation by light of a certain frequency or combination of frequencies. In a very real sense, color does not exist as a physical property, and it is technically incorrect to think of objects as “having” a specific color. Objects have spectral reflectance properties, which combined with the specific spectral characteristics of the light source illuminating them and the particular sensitivities of a given individual’s visual system combine to create the perception we call “color.4.Hence, the material of an object and the color of light that strikes that object determines the apparent color of that object.
