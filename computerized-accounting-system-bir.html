My opinion on having a computerized library system:

Computerized library system is a type of system which caters and performs libraries operations such as :  

borrowing and returning of books, adding and updating of users and information, adding and updating books and information, inventory of books and other library materials, browsing of library materials list to make searching more easier.

21. What are the parts and information found in computerized library System?


Answer:

1. Library Catalog: This is a database of all the items in the library's collection, including books, e-books, DVDs, CDs, and other physical items.

2. Electronic Resources: This includes online databases, digital databases, and other digital resources such as e-books, e-journals, and streaming media.

3. Online Reservation System: This is a system that allows users to reserve items in the library's collection, either online or in person.

4. Library Management System: This system enables the library staff to manage the library's collection, circulation, and user accounts.

5. Circulation System: This system allows users to check out and return items in the library's collection.

6. User Accounts: This includes the user's personal information, library card, and other information related to the user's account.

7. Search Engine: This is a search tool that allows users to search for items in the library's collection.

8. Reporting Tools: This includes tools for generating reports about the library's collection and users.
